# PSP恋愛アドベンチャーゲーム『AMNESIA(アムネシア)』と『エミル・クロニクル・オンライン』のコラボレーションくじ発売！

## introduction

とある架空世界の、とある架空の国の、とある架空の街の物語。

それは8月1日のこと。

朝目覚めた主人公は８月１日以前の記憶を『全て』失ってしまっていた……。

自分がどのような人生を送ってきたかも、周囲の人間関係もまっさら白紙の状態。

そんな彼女の目の前に現れたのは、自身を『精霊』と名乗るオリオンという少年だった。

オリオンの導きにより、記憶を取り戻すために奮闘することになる主人公。

まずは、自分の日常を知ろうとした矢先、彼女のものであるらしい携帯電話が着信を告げる。

――液晶画面に表示されたのは、やはり『知らない』名前だった――

## AMNESIA(アムネシア)とは

突然すべての記憶を失ってしまった主人公。彼女のもとに「恋人」と名乗る青年が現れる。ゲームはある事情で記憶をなくした事を話せないまま彼と接しつつ記憶を取り戻していく恋愛アドベンチャーゲーム。

注目は、原画を担当した花邑まい氏の美麗なイラストと、恋人に誰を選ぶかにより、各キャラの世界はパラレルワールドになっており、登場キャラクターの関係性も変化するストーリーの多様性だろう。

個性の強い青年たちとの恋愛模様を堪能できるゲームとなっている。
