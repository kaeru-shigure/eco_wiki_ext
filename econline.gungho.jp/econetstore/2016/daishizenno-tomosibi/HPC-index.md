◆「ポッケ・レタラペ（赤・上）」「キツネのもふもふマフラー（白）」「ポッケ・レタラペ（赤・下）」「テタラヘ・レタラペ（紺・上）」「テタラヘ・レタラペ（紺・下）」「アンヌ・ムツベ（赤）」「おおかみのじゅうたん（茶）」「リスのぬいぐるみ（茶）」について

アップタウン「裁縫おばさん」の家にいる「染色娘」に上記のアイテムを渡すと、各アイテムをそれぞれ決められた別の色に染め直したり、別のアイテムに交換することができます。  
また、染め直したアイテムを「染色娘」に渡すと、元のアイテムに戻すことができます。

【ご注意】  
融合している場合は染色・交換ができません。  
染色・交換をすると、アイテムのイリスカードスロット数や強化値は初期値に戻ってしまいますのでご注意ください。また、スロットに装着していたイリスカードは消滅してしまいますのでご注意ください。

パートナー憑依について

「アカリの守護魔ハート」または「守護魔・アカリ」を所持しているか装備している時に、アイテム情報ウインドウの「憑依」ボタンをクリックすると、キャラクターの代わりに自分の装備アイテムに憑依させることができます。  
「アカリの守護魔ハート」または「守護魔・アカリ」を自分の装備アイテムに憑依させると、「アカリの守護魔ハート」または「守護魔・アカリ」がキャラクターの背後に出現します。  
憑依されたアイテムを装備しているキャラクターは、ステータスが上昇し、「アカリの守護魔ハート」「守護魔・アカリ」のスキル「業火爆」を使用することができるようになります。

パートナー憑依について

「コロボックル」を所持しているか装備している時に、アイテム情報ウインドウの「憑依」ボタンをクリックすると、キャラクターの代わりに自分の装備アイテムに憑依させることができます。  
「コロボックル」を自分の装備アイテムに憑依させると、「コロボックル」がキャラクターの背後に出現します。  
憑依されたアイテムを装備しているキャラクターは、ステータスが上昇します。

守護魔・アカリのスキル一覧

|  |  |
| --- | --- |
| ▲業火爆 | 対象に斬撃を行った後、範囲攻撃を発動させ、持続ダメージと、一定時間、物理防御力、魔法防御力を低下させる状態異常を与える。 |
| ▲内心炎 | 対象の武器に炎の力を宿らせる。 |
| ▲炎魂 | 一定時間、自分の物理最大攻撃力を上昇させる。 対象が状態異常の場合、ダメージが上昇する。 ※スキル対象が＜自分＞の条件でしか発動しません。 |
| ▲火封印 | 火属性の攻撃を受けたときダメージを軽減する。 |
| ▲吸炎 | 対象の武器に、その武器が与えたダメージの一部を持ち主がＨＰとして吸収できる効果を与える。 |

今回のECOドリームのテーマは「大自然の灯火」。毎月新しい物語が楽しめるイベント「星を守る者」に登場する守護魔をイメージしたアイテムをご用意いたしました！

今回のECOドリームには、「星を守る者」に登場する「アカリの守護魔ハート」がパートナーになって登場！  
「アカリの守護魔ハート」をパートナーにすると、「アカリの守護魔ハート」をキャラクターの代わりに自分に憑依させることができます。

さらに、「アカリの守護魔ハート」を右クリックすることで、「アカリの守護魔ハート」のセリフの一部を好きなセリフにカスタマイズすることができます。

また、北方に隠れ住むと言われる小人「コロボックル」がパートナーとして登場！ 「コロボックル」もキャラクターの代わりに自分に憑依させることができます。

「アカリの紹介状」をヘアサロンに持っていけば、男女どちらでも「アカリの守護魔ハート」とおそろいの髪型に変更可能。  
さらに、オプションで耳なし・耳付きを選択可能です。

「ポッケ・レタラペ（赤・上）」「キツネのもふもふマフラー（白）」「ポッケ・レタラペ（赤・下）」「テタラヘ・レタラペ（紺・上）」「テタラヘ・レタラペ（紺・下）」「アンヌ・ムツベ（赤）」「おおかみのじゅうたん（茶）」「リスのぬいぐるみ（茶）」は、アップタウン「裁縫おばさん」の家にいる「染色娘」に渡すと別の色に染め直したり、別のアイテムと交換することができます。

B賞とC賞にはイベント「星を守る者」などで使える「ＥＸイベントチケット」が、D賞とE賞には、使用すると一定時間パートナーの信頼度が上昇しやすくなる「みんなのおやつ（初級）」がついてお得！

ECOドリーム「大自然の灯火」で「星を守る者」の世界観を楽しんじゃおう！
