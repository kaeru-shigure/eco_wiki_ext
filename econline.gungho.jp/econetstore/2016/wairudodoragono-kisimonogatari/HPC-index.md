友情の一撃について

「ワイルドドラゴ・アルマ」がパートナーの時に、スキルウインドウのベースタブに「友情の一撃」が追加されます。  
「友情の一撃」を実行すると、「ワイルドドラゴ・アルマ」が狂戦士状態になります。  
※通常の狂戦士状態とは異なり、思考が強制的に切り替らず、ＡＩカスタマイズ等のスキルが使用可能です。  
既に狂戦士状態だった場合は狂戦士状態が解除されます。  
運搬演習では使用できません。

ワイルドドラゴ・アルマのスキル一覧

|  |  |
| --- | --- |
| ▲龍章鳳姿 | 相手を凄い力で攻撃する。 |
| ▲荒竜の逆鱗 | 怒りを溜めこみダメージを受けるごとに攻撃力が上がる。 |
| ▲騎士道の加護 | 対象とその周囲の防御力と魔法防御力を上昇させ、クリティカルダメージ軽減効果を付与する。 |
| ▲騎士道の誇り | 自身中心の範囲の対象に最大HPを上昇させるバフを付与する。 |

連動モーションについて

「ワイルドドラゴ・アルマ」がパートナーの時に、エモーションウインドウのモーションタブに「連動モーション」が追加されます。  
「連動モーション」を実行すると、キャラクターとパートナーが二人同時に特別なモーションを実行します。  
「連動モーション」はキャラクターとパートナーが静止状態の時に実行されます。  
移動時や戦闘時は実行されません。

今回のECOドリームのテーマは「ワイルドドラゴの騎士物語」。毎月新しい物語が楽しめるイベント「神魔の来訪」に登場するアルマをイメージしたアイテムをご用意いたしました！

今回のECOドリームには、「神魔の来訪」に登場する「ワイルドドラゴ・アルマ」がパートナーになって参上！

「ワイルドドラゴ・アルマ」をパートナーにするとスキルに「友情の一撃」が追加され、パートナーを狂戦士状態にすることができます。  
また、モーションに「パートナー連動モーション」が追加され、キャラクターと一緒に特別なモーションを実行することができます。  
さらに、「ワイルドドラゴ・アルマ」を右クリックすることで、「ワイルドドラゴ・アルマ」のセリフの一部を好きなセリフにカスタマイズすることができます。

「城壁のある要塞古城」は飛空庭の家で、騎士の戦場となりそうな城壁付きの要塞古城です。

「ワイルドドラゴの紹介状」をヘアサロンに持っていけば、男女どちらでも「ワイルドドラゴ・アルマ」とおそろいの髪型に変更可能。

「竜騎士の鎧（赤）」「竜騎士のマント（赤）」「竜騎士のハーフブーツ（赤）」「竜騎士の斧（赤）」「竜騎士の角飾り（緑）」「マーセナリーベスト（白）」「マーセナリーパンツ（白）」「城壁（グレー）」「大きな壁掛け燭台（黒）」は、アップタウン「裁縫おばさん」の家にいる「染色娘」に渡すと別の色に染め直したり、別のアイテムと交換することができます。

B賞とC賞にはイベント「神魔の来訪」などで使える「ＥＸイベントチケット」が、D賞とE賞には、一定時間パートナーの信頼度獲得が上昇する「みんなのおやつ（初級）」がセットでついてお得！
