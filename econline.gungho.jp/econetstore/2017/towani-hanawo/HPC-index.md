◆「桜の足袋（白）」「悠久ノ和装束（桃）」「悠久ノ下駄（黄色）」「おしゃれ和装ドレス（緑）」「優雅な着流し（緑）」「悠久ノ杖（桃）」について

アップタウン「裁縫おばさん」の家にいる「染色娘」に上記のアイテムを渡すと、各アイテムをそれぞれ決められた別の色に染め直したり、別のアイテムに交換することができます。  
また、染め直したアイテムを「染色娘」に渡すと、元のアイテムに戻すことができます。

【ご注意】  
融合している場合は染色・交換ができません。  
染色・交換をすると、アイテムのイリスカードスロット数や強化値は初期値に戻ってしまいますのでご注意ください。また、スロットに装着していたイリスカードは消滅してしまいますのでご注意ください。

パートナー憑依について

「トワの守護魔ハート」または「守護魔・トワ」を所持しているか装備している時に、アイテム情報ウインドウの「憑依」ボタンをクリックすると、キャラクターの代わりに自分の装備アイテムに憑依することができます。  
「トワの守護魔ハート」または「守護魔・トワ」が自分の装備アイテムに憑依すると、「トワの守護魔ハート」または「守護魔・トワ」がキャラクターの背後に出現します。  
憑依されたアイテムを装備しているキャラクターは、ステータスが上昇し、「トワの守護魔ハート」「守護魔・トワ」のスキル「幸ノ魂」を使用することができるようになります。

トワの守護魔ハート（守護魔・トワ）のスキル一覧

|  |  |
| --- | --- |
| ▲死魅霊還 | 対象にアンデット属性になる魔法攻撃を行った後、 アンデッドに対して効果の高いＨＰ吸収攻撃を行い、 自分のＨＰとＭＰを回復する。 |
| ▲桜花爛漫 | 一定時間、自身とその周囲のＨＰと基本的な状態異常を一定間隔で回復するオーラを纏う。 ※スキル対象が＜自分＞の条件でしか発動しません。 ※効果中、透明化などの特殊効果が解除されます。 |
| ▲適者生存 | 対象にトランスボディ効果を付与する。 トランスボディ中、火、水、風、土属性攻撃を受けると、その属性耐性を得ることができる。 ファイアシールド、ウィンドシールド、アイスシールド、アースシールドの習得LVにより効果が更に高まる。 ※ただし、地面設置系魔法移動系魔法は適用されません。 |
| ▲仮初ノ命 | スキルの効果中残りＨＰ以上のダメージを１度だけ敵にそのまま反射する仮初の魂を纏う。 ※一部反射できないダメージ有り。 ※黒薔薇の棘で反射したダメージは仮初ノ命では反射することができない。 ※スキル対象が＜自分＞の条件でしか発動しません。 |
| ▲幸ノ魂 | 効果時間中、消費ＭＰと消費ＳＰを軽減する。 ※スキル対象が＜自分＞の条件でしか発動しません。 |

今回のECOドリームのテーマは「トワに花を」。毎月新しい物語が楽しめるイベント「星を守る者」に登場する守護魔をイメージしたアイテムをご用意いたしました！

今回のECOドリームには、「星を守る者」に登場する「トワの守護魔ハート」がパートナーになって登場！

「トワの守護魔ハート」をパートナーにすると、「トワの守護魔ハート」をキャラクターの代わりに自分に憑依させることができます。  
さらに、チェンジで「守護魔・トワ」と入れ替えることができます。

また、春の訪れを告げる小さな妖精「春の妖精」が胸アクセサリーとして登場！

「トワの紹介状」をヘアサロンに持っていけば、男女どちらでも「トワの守護魔ハート」とおそろいの髪型に変更可能。

「桜の足袋（白）」「悠久ノ和装束（桃）」「悠久ノ下駄（黄色）」「おしゃれ和装ドレス（緑）」「優雅な着流し（緑）」「悠久ノ杖（桃）」は、アップタウン「裁縫おばさん」の家にいる「染色娘」に渡すと別の色に染め直したり、別のアイテムと交換することができます。

B賞とC賞にはイベント「星を守る者」などで使える「ＥＸイベントチケット」が、D賞、E賞には使用するとパートナーの信頼度が上昇する「信頼度あがーる（小）」がついてお得！

ECOドリーム「トワに花を」で「星を守る者」の世界観を楽しんじゃおう！
