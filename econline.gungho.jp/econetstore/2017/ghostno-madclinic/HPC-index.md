パートナー憑依について

「ゴースト・アルマ」を所持しているか装備している時に、アイテム情報ウインドウの「憑依」ボタンをクリックすると、キャラクターの代わりに自分の装備アイテムに憑依させることができます。  
「ゴースト・アルマ」を自分の装備アイテムに憑依させると、「ゴースト・アルマ」がキャラクターの背後に出現します。  
憑依されたアイテムを装備しているキャラクターはステータスが上昇します。

ゴースト・アルマのスキル一覧

|  |  |
| --- | --- |
| ▲プレメディ | 対象に無数の攻撃を浴びせる。 |
| ▲トランキライザー | 対象を中心に範囲攻撃を行い、一定時間、次の行動に移るまでの動作を遅らせる。 |
| ▲リターンボディ | 被ダメージの一部をＨＰ、ＳＰとして吸収する。 |
| ▲ホーンテッドコール | 範囲内にいる対象を攻撃しつつ、ステータス上昇効果を全て解除する。 ※スキル対象が＜自分＞の条件でしか発動しません。 |
| ▲グッドメディスン | 自身の攻撃の後に追撃が発生する。 稀に強力な追撃が発生する事がある。 ※スキル対象が＜自分＞の条件でしか発動しません。 |

連動モーションについて

「ゴースト・アルマ」がパートナーの時に、エモーションウインドウのモーションタブに「連動モーション」が追加されます。  
「連動モーション」を実行すると、キャラクターとパートナーが二人同時に特別なモーションを実行します。  
「連動モーション」はキャラクターとパートナーが静止状態の時に実行されます。  
移動時や戦闘時は実行されません。

ハザードフィーバーについて

「ゴースト・アルマ」がパートナーの時に、スキルウインドウのベースタブに「ハザードフィーバー」が追加されます。  
「ハザードフィーバー」を実行すると、キャラクターが「ゴースト・アルマ」と一緒にモーションを実行し、自分を中心とした範囲内の味方を一定時間ノックバックしなくさせ、少量物理防御力を上昇させます。

また、傷病者を運ぶ「ストレッチャー」がライドパートナーとして登場！
