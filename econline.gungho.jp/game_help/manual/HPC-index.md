## 画面のみかた

### ゲーム画面

ゲームの基本画面です。

| --- | --- | --- | --- |
| 1 | メインウィンドウ | Ctrl+z | キャラクターの名前、レベル、HP、MPなどが表示されます。 |
| 2 | ショートカットウィンドウ |  | ファンクションキーにアイテムやスキルなどを登録し、 |
| 3 | ミニマップ | Ctrl+m | キャラクター周辺のマップが表示されます。 |
| 4 | 職業チャットログウィンドウ |  | 職業（JOB）ごとのチャットログです。 |
| 5 | ログウィンドウ |  | チャットでの発言や、ゲーム内アナウンスなどが表示されます。 |
| 6 | チャットウィンドウ |  | チャットでの発言を入力するためのウィンドウです。 |
| 7 | アイコンメニュー |  | 各種のウィンドウを表示するためのメインメニューです。 |
| 8 | 属性アイコン |  | キャラクターの攻撃・防御属性が表示されます。 |

#### 1.メインウィンドウ

メインウィンドウでは自分のキャラクターの情報が確認できます。

| 表示項目 | 説明 |
| --- | --- |
| LV（現在のレベル） | 現在のキャラクターのレベルです。 |
| JOB（現在の職業レベル） | 現在キャラクターの職業レベルです。 |
| EXP（基本経験値） | ・次のレベルまでに必要な経験値です。  ・100％になるとレベルがひとつあがります。 |
| JOBEXP（職業経験値） | ・次の職業レベルまでに必要な経験値です。  ・100％になるとレベルがひとつあがります。 |
| HP（ヒットポイント） | ・キャラクターの生命力です。  ・キャラクターを座らせたり、アイテムを使うことで回復します。 |
| MP（マジックポイント） | ・魔法スキルを使うためのポイントです。  ・キャラクターを座らせたり、アイテムを使うことで回復します。 |
| SP（スタミナポイント） | ・物理スキルを使うためのポイントです。  ・キャラクターを座らせたり、アイテムを使うことで回復します。 |
| EP（エンゲージポイント） | ・H.E.ARTスキルを使うためのポイントです。  ・プレイヤーの行動によって回復します。 |
|  |

![](../../manual/_img/ind_img_02.jpg)

#### 2.ショートカットウィンドウ

ショートカットウィンドウでは、キーボードのF1～F12までのファンクションキーに、スキルやアイテムを登録することができます。

![](../../manual/_img/ind_img_03.jpg)

#### 6.チャットウィンドウ

ゲーム内のほかのキャラクターたちとチャットを楽しむことができます。

![](../../manual/_img/ind_img_04.jpg)

| アイコン | チャット種別 | チャットの発言範囲 |
| --- | --- | --- |
|  | オープンチャット | キャラクターの周辺にいるほかのキャラクター全て。 |
|  | パーティチャット | キャラクターが加入しているパーティーのメンバーのみ。 |
|  | リングチャット | キャラクターが加入しているリングのメンバーのみ。 |
|  | ウィスパーチャット | 指定した特定のキャラクターのみ。 |
|  | チャットルームチャット | キャラクターが参加しているチャットルームのメンバーのみ。 |
|  | ミニゲームチャットルーム | ミニゲームに参加しているメンバーのみ。 |
|  | 職業チャット | 特定の職業分野のみ。 |

![](../../manual/_img/ind_img_05.jpg)

#### 7.アイコンメニュー

各種のウィンドウを表示するためのメニューです。

| アイコン | 名称 | 説明 |
| --- | --- | --- |
|  | キャラクター | ステータスやスキル、装備などのキャラクターの情報と装備しているパートナーの情報を確認することができます。 |
|  | 持ち物 | 所持しているアイテムの一覧を表示します。 |
|  | コミュニティ | フレンドやリング、パーティーの情報や募集コメントリスト、メールなどを確認することができます。 |
|  | クエスト・ナビインフォ | クエストウィンドウやイベントナビゲーション、旅人のメモを開くことができます。 |
|  | 憑依 | 自身に憑依したり、装備している憑依やたまいれを外すことができます。また、憑依カタログもこちらから開くことができます。 |
|  | ECOストア | ECO SHOPを開くことができます。また、現在発売中のECOくじの情報を見ることができます。 |
|  | システム | ログアウトはこちらから行えます。 |
