## システム

### イリスカードポスター　【いりすかーどぽすたー】

![](../../gamedata/system/_img/iriscard_poster/sys_iriscard_poster_img_01.jpg)

イリスカードを特殊な額縁に入れることで、飛空庭や飛空城に家具として設置することができる。

【ポイント】

- 所持しているイリスカードをダウンタウンの「カードラボ」にいるNPC「額縁職人」より購入できる特殊な額縁に入れることで、家具アイテムにすることができます。

#### イリスカードをポスターにしよう！

##### 専用の額縁を入手

イリスカードをポスターにするには、ダウンタウンの「カードラボ」にいるNPC「額縁職人」にGoldを支払うことで購入できる「イリスカード専用の額縁（Ｌ）」か「イリスカード専用の額縁（Ｓ）」が必要になります。

![](../../gamedata/system/_img/iriscard_poster/sys_iriscard_poster_img_02.jpg)

|  |  |
| --- | --- |
| イリスカード専用の額縁（Ｌ） | イリスカード専用の額縁（Ｓ） |
| イリスカード専用の額縁（Ｌ） | イリスカード専用の額縁（Ｓ） |

初回のみ各額縁をそれぞれ1個もらうことができます。

##### 額縁を設置してイリスカードを差し込む

飛空庭・飛空城に額縁を設置し、額縁を右クリックして「使う」を選ぶと、インベントリの中にあるイリスカードが表示されます。

![](../../gamedata/system/_img/iriscard_poster/sys_iriscard_poster_img_03.jpg)

![](../../gamedata/system/_img/iriscard_poster/sys_iriscard_poster_img_05.jpg)

##### 額縁のイリスカードを外す

![](../../gamedata/system/_img/iriscard_poster/sys_iriscard_poster_img_04.jpg)

額縁を設置し、額縁を右クリックして「使う」を選ぶと、差し込んでいるイリスカードを取り外してインベントリに戻すことができます。

飛空庭・飛空城に設置してある額縁は、持ち主のみイリスカードの変更が出来ます。
