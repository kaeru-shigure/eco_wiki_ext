## システム

### まいにちダンジョン【まいにちだんじょん】

![まいにちダンジョン](../../gamedata/system/_img/daily-dungeon/sys_d-dungeon_img_01.jpg)

「気軽にダンジョンに挑戦したい！」 「パートナーとの狩りをもっともっと楽しみたい！」という方におススメのダンジョン。

メニューから選択するだけで、すぐにダンジョンに挑戦することができ、様々な報酬を入手できます。

LV50以上のキャラクター推奨となります。

【ポイント】

- 「経験値ダンジョン」「信頼度ダンジョン」「ゴールドダンジョン」の3種類から入場するダンジョンを選べます。
- AM4:00～4:59の間は「まいにちダンジョ」ンに入場することができません。
- DEM以外の種族は「パートナー」か「ライドパートナー」を装備していないと、「まいにちダンジョン」に入場できません。
- DEM種族は一部報酬が異なります。

#### まいにちダンジョンに入る

##### まいにちダンジョンに入る

「まいにちダンジョン」には、メニューバーからクエストナビインフォを開き、「まいにちダンジョン」をクリックするだけで入場できます。

![まいにちダンジョンに入る](../../gamedata/system/_img/daily-dungeon/sys_d-dungeon_img_02.jpg)

騎士団演習・都市攻防戦・エンシェントアーク各キャプチャー・インスタンスダンジョンやイベント中などの一部のMAPからは入場できません。

#### まいにちダンジョンの種類

「まいにちダンジョン」には「経験値」「信頼度」「ゴールド」の三種類があり、それぞれが「初級」と「上級」に分かれています。

- 毎日AM5:00に、ダンジョンの初回クリア時の報酬受取状況がリセットされます。
- ダンジョン内のモンスターからは、経験値・職業経験値・アイテム・信頼度などは入手できません。

#### まいにちダンジョンで手に入るもの

「まいにちダンジョン」内の最深部にいるNPC「ダンジョンのヌシ」に話しかけることで、様々な報酬を入手できます。

|  |  |
| --- | --- |
| ①ダンジョン報酬 | 「信頼度」 「経験値」「ゴールド」などダンジョンごとの報酬がもらえます。 |
| ②まいにちＢＯＸ | 新米冒険者におすすめの「装備品」や、「家具」「パートナー」、「強化結晶」「ブランクイリスカード」などをはじめとした様々なアイテムが入っている「まいにちＢＯＸ」がもらえます。 |
| ③まいにち報酬（毎日初回クリア時） | 1日1回、「∽経験値ふえーる５０（１０分）」や「∽プライベートダンジョンつくーる（１０分）」などの、狩りに便利なアイテムのいずれかがもらえます。 |

![まいにちダンジョンで手に入るもの](../../gamedata/system/_img/daily-dungeon/sys_d-dungeon_img_03.jpg)

| ダンジョンの種類 | ダンジョン報酬（毎日初回クリア時） | ダンジョン報酬（毎日二回目以降クリア時） |
| --- | --- | --- |
| 経験値ダンジョン | 経験値（初回のみ取得量増加）＋パートナー経験値（初回のみ取得量増加） | 経験値（通常）＋パートナー経験値（通常） |
| まいにちＢＯＸ１ | まいにちＢＯＸ１ |
| まいにち報酬 | － |
| 信頼度ダンジョン | ダンジョン報酬 | ダンジョン報酬 |
| 信頼度（初回のみ取得量増加） ※DEMの場合はEP（初回のみ取得量増加） | 信頼度（通常） |
| まいにちＢＯＸ２ | まいにちＢＯＸ２ |
| まいにち報酬 | － |
| ゴールドダンジョン | ダンジョン報酬 Gold（初回のみ取得量増加） | ダンジョン報酬 Gold（通常） |
| まいにちＢＯＸ３ | まいにちＢＯＸ３ |
| まいにち報酬 | － |

- 得られる経験値や信頼度は、キャラクター／パートナーのＬｖ・信頼度に合わせて変わります。
- 入手できる「まいにちＢＯＸ」はダンジョン別となり、開封時に入手できるアイテムが異なります。
- 入手できる「まいにち報酬」は選べません。また、「まいにち報酬」のアイテムは、入手から24時間で消滅いたしますのでご注意ください。

#### まいにちダンジョンからの転送

「まいにちダンジョン」内の最深部にいるNPC「ダンジョンのヌシ」は、元の場所以外に、いくつかのダンジョンの入り口まで転送してくれます。

【転送先のダンジョン】

- 大陸の洞窟入り口
- 氷結の坑道入り口
- 廃炭鉱入り口
- ノーザンダンジョン入り口
- 神託の風穴入り口
- アンデッド城入り口

![まいにちダンジョンからの転送](../../gamedata/system/_img/daily-dungeon/sys_d-dungeon_img_04.jpg)

- 他のダンジョンへの転送は、エミル世界から「まいにちダンジョン」に入場した時のみとなります。
