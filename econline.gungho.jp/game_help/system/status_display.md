## システム

### 状態表示アイコンカスタマイズ機能【じょうたいひょうじあいこんかすたまいずきのう】

![](../../gamedata/system/_img/status_display/sys_sta_img_01.jpg)

自分のキャラクターの発生している様々なステータス変化の状態がわかる「状態表示」アイコンの表示優先度をカスタマイズすることができます。

【ポイント】

- 画面は以下の項目のように変更できます。

![](../../gamedata/system/_img/status_display/sys_sta_img_02.jpg)

#### カスタマイズ方法

![](../../gamedata/system/_img/status_display/sys_sta_img_03.jpg)

| No. | 名称画面の説明 |
| --- | --- |
| 1 | このリストで、このウィンドウに表示するアイコンの種類を指定できます。 |
| 2 | このチェックボックスをonにしておくと、アイコンは状態変化の発生順に左から表示されます。 |
| 3 | 表示の優先度LVを設定します。高いほど優先度が上がります。 |
| 4 | 「更新」をクリックすると、設定した優先度順にソートします。 |
| 5 | カスタマイズした設定に名前をつけて「セーブ」をクリックすると保存されます。また、以前セーブしたファイル名を選択し、「ロード」を押すとウィンドウに反映されます。 |

セーブしたカスタマイズ内容は、「\Emil chronicle online\data\statusicon」に保存されます。
