## システム

### ユニオンペットシステム【ゆにおんぺっとしすてむ】

![](../../gamedata/system/_img/unionpet/sys_uni_img_01.jpg)

主人とより深い心の結びつきを得たペット。

【ポイント】

- ユニオンペットの成長に伴い、ユニオンペットのステータスを上げることができます。
- 行動を指定したり、様々な新しいスキルを選んで覚えさせることができます。
- 専用の装備を身に着けることができます。特定のペット系統のみ装備可能なアイテムも存在します。
- 元のペットの成長具合によって強さが変化します。

#### ユニオンペット化

南アクロニア平原の「ザナフ」にからの依頼をクリアする事で、ユニオンペット化が可能になります。

ユニオンペット化したペットは、再度ザナフに話しかける事でペットに戻すことができます。

![](../../gamedata/system/_img/unionpet/sys_uni_img_02.jpg)南アクロニア平原のNPC「ザナフ」

#### ユニオンペットの行動

ユニオンペットの行動はAIで設定することができます。
AIは、あらかじめ特定の行動が設定してあるもののほかに、自分で行動条件を指定できる

![](../../gamedata/system/_img/unionpet/sys_uni_img_03.jpg)ユニオンペットを右クリックして「かんたんAI設定表示」を選ぶと、

##### かんたんAI設定

あらかじめ設定されたAIを選択するだけで、ユニオンペットの行動を設定できます。

| 攻撃重視のAI | | 主人重視のAI | | 追従のAI | |
| --- | --- | --- | --- | --- | --- |
|  | 一番近くに居るモンスターに攻撃スキルを優先して攻撃します。 |  | 主人がモンスターに攻撃されるまでまたは、自分がモンスターに攻撃されるまで行動を起こしません。 |  | 主人に自動でついてきてくれます。 |

##### AIカスタム設定

「AIカスタム設定」画面で、「条件」や「行動／スキル」パターンのスロットに「アクトキューブ」をセットすることで、AIをカスタマイズすることができます。「パッシブスキルアクトキューブ」は覚えた時点で自動的に発動します。

![](../../gamedata/system/_img/unionpet/sys_uni_img_10.jpg)ユニオンペットを右クリックして「AIカスタムウィンドウ表示」を選ぶと、AIカスタム設定画面が表示されます。

###### カスタムAIを実際に設定してみよう

例：攻撃支援重視のカスタムAI

ペット
:   背負い魔・ネコマタ（杏）

必要レベル
:   Lv1以上

必要信頼度
:   黒以上

|  | 条件 | スキル |
| --- | --- | --- |
| ① | 自分のＨＰ≦８０％＜自分＞ | ▲ヒーリング |
| ② | 主人のＨＰ≦５０％＜主人＞ | ▲ヒーリング |
| ③ | 主人の敵が ＨＰ≦９０％まで＜主人＞ | ▲ディレイキャンセル |
| ④ | 主人の敵が ＨＰ≦１００％まで＜主人＞ | ▲クリティカルアシスト |
| ⑤ | ― | ― |
| かんたんAI設定：「追従」 | | |

【解説】  
自分（ペット）のHPは80％を維持しつつ、プレイヤーのHPを50％以上に保ってくれます。  
プレイヤーがスキルで攻撃をすると、すかさずプレイヤーに支援魔法をかけてくれます。

「▲クリティカルアシスト」はアクトキューブ自販機で入手する事ができます。  
MP/SPがすぐ切れる場合は武器を持たせてみましょう

例：回復支援重視のカスタムAI

ペット
:   背負い魔・ネコマタ

必要レベル
:   Lv1以上

必要信頼度
:   黒以上

|  | 条件 | スキル |
| --- | --- | --- |
| ① | 主人のＨＰ≦０％＜主人＞ | ▲リザレクション |
| ② | 自分のＨＰ≦８０％＜自分＞ | ▲ヒーリング |
| ③ | 主人のＨＰ≦５０％＜主人＞ | ▲ヒーリング |
| ④ | 主人の敵が ＨＰ≦６０％まで＜主人＞ | ▲アタック・バースト |
| ⑤ | 主人の敵が ＨＰ≦８０％まで＜主人＞ | ▲ディフェンス・バースト |
| かんたんAI設定：「追従」 | | |

【解説】  
自分（ペット）のHPは80％を維持しつつ、プレイヤーのHPを50％以上に保ってくれます。  
プレイヤーのHPが0になったら、最優先でリザレクションを行い、それ以外の場面では支援魔法を使ってくれます。

支援重視型にする場合は、「魔法攻撃力を強く」して「精神力を鍛える」と効果的です。

##### アクトキューブ

ユニオンペットの行動条件や行動内容／スキルを指定するアイテムです。南アクロニア平原にある「アクトキューブ製造機」や「ECO SHOP」から購入、またはモンスターなどから入手することができます。

| アイコン | 内容 |
| --- | --- |
|  | 行動結果の対象が<主人>の行動条件のアクトキューブ |
|  | 行動結果の対象が<自分>の行動条件のアクトキューブ |
|  | 行動結果の対象が<仲間>の行動条件のアクトキューブ |
|  | 行動結果の対象が<敵>の行動条件のアクトキューブ |
|  | 行動内容のアクトキューブ |
|  | スキルのアクトキューブ |
|  | パッシブスキルのアクトキューブ |
|  | 条件数値とターゲットを細かく指定できる、行動条件のアクトキューブ |

| 条件・行動パターン（アクトキューブ）の習得 |
| --- |
| - ユニオンペットを装備して「アクトキューブ」を使用すると、ユニオンペットが条件・行動パターンを覚えます。 - 覚えた条件・行動パターンはAIカスタム設定画面に反映され、AI設定に利用できるようになります。 - 「条件」は30個、スキルや動作などの「行動」は20個まで覚えさせることができます。 |

![](../../gamedata/system/_img/unionpet/sys_uni_img_05.jpg)南アクロニア平原の「アクトキューブ製造機」

【ご注意】

- ※ペットがもともと習得しているスキルは、すでにアクトキューブとして覚えた状態でユニオンペットに引き継がれます。
- ※覚えたアクトキューブは、カスタマイズ画面上で右クリックし、「忘れる」を選択することで忘れることができます。
- ※忘れたアクトキューブはアイテムには戻りません。
- ※ユニオンペットを通常のペットに戻しても習得したアクトキューブは忘れません。
- ※ユニオンペット状態で地面に置いたり、露店やトレードなどで他のキャラクターに渡してしまうと、習得したアクトキューブやLVはすべて消失してしまいます。ご注意ください。

#### ユニオンペットの成長

ユニオンペットは、「ユニオンミール」を与えることで経験値を獲得し、LVアップします。

「ユニオンミール」にはいくつかの種類があり、  
南アクロニア平原の「マルク」から購入できるもの  
釣りや飛空城ファームの農作物と合成することで入手できるもの  
「ECO SHOP」から購入できるもの  
などがあります。

LVアップすると、「成長ポイント」が獲得できます。

- ユニオンペットのLVによって与えられるユニオンミールの種類が異なります。
- 一度に与えられるユニオンミールの量には制限があります。
- 一度ユニオンミールを与えると、次に与えるまでしばらく時間が必要になります。

![](../../gamedata/system/_img/unionpet/sys_uni_img_06.jpg)

##### ユニオンペットの強化

LVが上がるごとに獲得できる「成長ポイント」を振り分け、ステータスを上げることで、ユニオンペットを強くすることができます。

ユニオンペットの「系統」と対応する特定の「育成指針」に割り振った成長ポイントが規定数に達すると、成長ポイントを割り振って上昇するステータスとは別にステータスボーナスが加算されます。  
系統ボーナスが発生する「育成指針」は系統ごとに異なります。

また、ユニオンペット専用の武具を入手して装備すれば、さらにパワーアップさせることができます。

##### ステータスの低下

ユニオンペットはHPが0になると負傷状態となり、一定時間経過するまでステータスが下がった状態になります。

また、負傷状態のユニオンペットは、アイテムインベントリに戻ります。

- エクストラアイテム「Ｕ・成長指針忘れドリンク」を使用すると、成長ポイントをリセットすることができます。
- 負傷状態の時間はLVが上がるごとに延び、ログアウト中も経過します。
- 騎士団演習、メタモーバトル、闘技場のPvPではHPが0になっても負傷状態にはなりません。
- エクストラアイテム「Ｕ・すぐよくなーるドリンク」を使用すると、負傷状態をすぐに回復することができます。

![](../../gamedata/system/_img/unionpet/sys_uni_img_07.jpg)

#### ユニオンペットの成長

信頼度とは、ユニオンペットが主人とどれだけの時を過ごし、 どれだけの冒険をしてきたかを表すものです。  
信頼度はフレンドリスト内の「ユニオンペットリストウィンドウ」の信頼度アイコンの色で確認することができます。  
信頼度は一緒にモンスターと戦うことで上昇します。また、与える餌によって信頼度の上昇率が変わります。  
ユニオンペットの信頼度が上がると、そのユニオンペットのステータスにボーナスが付与されます。
また、信頼度が上がると特別なセリフをしゃべるようになります。

![](../../gamedata/system/_img/unionpet/sys_uni_img_08.jpg)

戦闘不能になると、親密度が減少しますが信頼度は減少しません。

#### ブリーダーの特典及び既存のペット関連スキル

ペット能力を向上させるタイプのスキルは、通常のペットほどではありませんがユニオンペットにも一定の効果があります。

また、他にも特定のペットにスキルを使わせることができるユーザースキルは、 ユニオンペットにも問題なく使用することができます。

ジョブがブリーダーの場合、信頼度ボーナス、系統ボーナスとは別に、ユニオンペットのステータスがアップします。

【その他注意点】

- ユニオンペット状態では倉庫に入れることができません。また、「売却・トレード」をすることはできません。
- ユニオンペット状態では飛空庭に配置することができません。
- ユニオンペット状態ではペットを転生させることはできません。

- ※「背負い魔・ネコマタ（新緑）」の様に、モデル変更を伴う転生をさせた場合、転生前のユニオンペット情報を引き継ぐため、  
  　　転生後のペットに追加される固有のスキルは所持していません。
