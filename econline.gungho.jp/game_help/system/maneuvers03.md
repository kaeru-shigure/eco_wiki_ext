### 一般演習【いっぱんえんしゅう】

#### ルール

![](../../gamedata/system/_img/maneuvers/sys_man03_img_01.jpg)

敵軍のキャラクターを倒したり、敵軍のシンボルを破壊したり、「戦場の輝き」を自軍の陣地に運ぶと入手できるポイント数を競う演習です。

#### ポイントについて

##### 戦場の輝きを納品する

フィールドの「エメラルド」「ガーネット」などの魔法物質は「戦場の輝き」をドロップすることがあります。

![戦場の輝き](../../gamedata/system/_img/maneuvers/sys_man03_img_08.jpg)

![戦場の輝きを納品する](../../gamedata/system/_img/maneuvers/sys_man03_img_09.jpg)

※ワールド対抗戦MAPでは各「騎士団隊長」は登場しません。各ワールドの「係員」に「戦場の輝き」を納品します。

##### 騎士団演習ワールド対抗戦の特殊ルール

騎士団演習ワールド対抗戦では、NPCに「戦場の輝き」を納品時に、個数に応じて特殊なアイテムをもらうことができます。

#### 戦場の輝きによる特殊効果

戦場の輝きを一定数納品すると、獲得できるポイントが増えたりなど自軍に有利になる様々な特殊効果が発動します。

中には、敵軍に対して効果を及ぼすものも存在します。

![特殊効果](../../gamedata/system/_img/maneuvers/sys_man03_img_11.gif)

#### ヒーロー状態

トルマリンを刈ったキャラクターは「ヒーロー状態」になり、以後、獲得する得点が通常の2倍になります。

ヒーロー状態のキャラクターがログアウトした場合、ヒーロー状態はその周囲にいるキャラクターにランダムで移ります。周囲に誰もいなかった場合、ヒーロー状態は消滅します。

ヒーロー状態のキャラクターを倒すと、倒したキャラクターにヒーロー状態が移ります。

トルマリンの出現と同時に全プレイヤーに出現位置を知らせる矢印が表示されます。  
ヒーロー状態の場合は、他のプレイヤーに憑依することはできません。

![ヒーロー状態](../../gamedata/system/_img/maneuvers/sys_man03_img_12.jpg)

#### 一般演習の特典

騎士団演習中の活躍に応じて「獲得経験値」「獲得職業経験値」「獲得CP」を入手できます。

また、活躍に応じた様々な特別賞や、演習中の獲得ポイントや行動不能になった回数などを踏まえて最終的な特典が決まります。

※演習中に獲得ポイントを一定以上稼がないと入手できません。

![一般演習の特典](../../gamedata/system/_img/maneuvers/sys_man03_img_13.jpg)

| 特別賞の種類 | 内容 |
| --- | --- |
| ブービーofデッド | すべての参加者中、行動不能になった回数が2番目～11番目に多かったキャラクター10名に贈られます。 |
| ナイチンゲール賞 | すべての参加者中、他のキャラクターを回復した量がもっとも多かったキャラクター上位10名に贈られます。 |
| 戦場の収集家 | すべての参加者中、魔法物質のドロップするアイテム「戦場の輝き」を納品した数がもっとも多かったキャラクター上位10名に贈られます。 |
| 隠れん坊賞 | すべての参加者中、演習中に受けたダメージがもっとも少なく、かつ魔法物質を10個以上破壊したキャラクター上位10名に贈られます。 |
| 暴れん坊賞 | すべての参加者中、多くの敵軍キャラクターに対して最初にダメージを与えたキャラクター上位10名に贈られます。 |
| スーパーマン賞 | すべての参加者中、瞬間最大被ダメージがもっとも多かったキャラクター上位10名に贈られます。 |

1つの賞を獲得できるのは最大で10名までです。  
同じ順位に同率のキャラクターが複数いて10名の枠を超えてしまう場合、その順位以降は「ノーコンテスト（無効）」となります。
