## システム

### H.E.ARTシステム　【はーとしすてむ】

![](../../gamedata/system/_img/heartskill/sys_hea_img_01.jpg)

冒険者同士の「人と人とのつながり」による絆を使い、様々な力を発揮する新たなコミュニケーションの形。

【ポイント】

- ログインやあいさつなど、コミュニケーションをすることで「EP（Engage Point）」を貯めることができます。
- EPを貯めることで、他のキャラクターと協力して特殊な「H.E.ARTスキル」を発揮することができます。
- ＤＥＭはEPを消費することで様々なカスタマイズをすることができます。

#### EP（Engage Point）

EPはキャラクターの行動によって貯まっていき、「H.E.ARTスキル」の使用やＤＥＭのカスタマイズにより消費されます。

EPはメインウィンドウで確認することができます。数値は「現在EP / 最大EP」を表しています。

EPの増減はキャラクターごとに行われるため、ゲームアカウント内で共通ではありません

![](../../gamedata/system/_img/heartskill/sys_hea_img_02.jpg)

##### EPを貯める

EPはゲーム内で以下の行動を行うことによって貯めることができます。

ただし、その行動を行った時に最大値をオーバーする場合は切り捨てとなります。

| 行動 | 増加ポイント |
| --- | --- |
| ログインする | 10ポイント（24時間に1回のみ） |
| あいさつする | 登録フレンド数に応じて変動（1日に1回のみ） |
| EP専用クエストを達成する | 30ポイント |
| 通常クエストを達成する | 3ポイント |

- EPの最大値は、リングに所属していない状態で30ポイントとなります。
- リングに所属することで、リングメンバーの人数に応じて最大値を110ポイントまで増やすことができます。

![](../../gamedata/system/_img/heartskill/sys_hea_img_03.jpg)
相手を右クリックし、「あいさつをする」を選ぶとあいさつがでｊきます。

#### H.E.ARTスキルを使う

EPを消費することで、仲間と力を併せた時のみ使うことができる特殊な「H.E.ARTスキル」を発動できます。

使用できるH.E.ARTスキルは職業の系統ごとに異なります。

![](../../gamedata/system/_img/heartskill/sys_hea_img_04.jpg)

##### H.E.ARTスキルを発動する

使用したいH.E.ARTスキルを選び発動すると、支援待ち状態になります。

他のキャラクターから、発動支援H.E.ARTスキル「受け取れ！」を受けると、H.E.ARTレベルゲージが変化します。

発動者の頭上のタイムゲージがフルになるとスキルが発動します。

- H.E.ARTレベルゲージの値によって、スキルの威力が変化します。
- 「H.E.ART レベル」の最大値はフレンド数で3～7ポイントまで増減します。
- 誰も支援者がいない場合は、スキルは失敗となり、EPは消費されません。

![](../../gamedata/system/_img/heartskill/sys_hea_img_05.jpg)

##### H.E.ARTスキルを支援する

他のキャラクターがH.E.ARTスキルを発動し、支援待ち状態になっている時に、
発動支援H.E.ARTスキル「受け取れ！」を使用すると、H.E.ARTスキルの発動を支援できます。

H.E.ARTスキルを発動しようとしている人を見かけたら、手伝ってあげよう。

![](../../gamedata/system/_img/heartskill/sys_hea_img_06.jpg)

#### ＤＥＭのカスタマイズ

EPを消費することで、ＤＥＭの成長や様々なカスタマイズを行うことができます。

ＤＥＭについて詳しくは[こちら](../../game_help/race/dem.html)をご参照ください。
