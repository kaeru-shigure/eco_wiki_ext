## システム

### リーダー戦演習【りーだーせんえんしゅう】

#### ルール

ランダムに選出される相手軍の「リーダー」や護衛のキャラクターを倒したり、フィールドの魔法物質を破壊すると入手できるポイント数を競う演習です。

![ルール](../../gamedata/system/_img/maneuvers/sys_man02_img_01.jpg)

#### リーダーについて

演習場にいる各軍のメンバーからそれぞれランダムでリーダーが選出されます。

![リーダーについて](../../gamedata/system/_img/maneuvers/sys_man02_img_02.jpg)

![全軍突撃](../../gamedata/system/_img/maneuvers/sys_man02_img_03.gif)

・リーダーになれるキャラクターが演習場にいない場合、ペナルティとして一定のポイントを失います。  
　また、リーダーがログアウトした場合は、ペナルティとして、所持ポイントからそれぞれ10％のポイントを他の軍に振り込みます。  
・リーダーに選ばれたキャラクターは憑依することができません。  
・リーダーの戦闘パラメータのボーナスは、ペットと騎乗ペットには適用されません。

#### ポイントについて

##### 敵軍キャラクターの討伐、魔法物質・シンボルの破壊

敵軍のキャラクターを倒したり、騎士団演習マップに出現する魔法物質や敵軍のシンボルを破壊することでポイントを入手できます。

| 敵軍のリーダーを倒す | リーダー以外のキャラクターを倒す | 各敵軍シンボル |
| --- | --- | --- |
| 敵軍のキャラクターを倒す | 敵軍のキャラクターを倒す | 各敵軍シンボル |
| 倒したリーダーの軍から30％のポイントを奪います。  倒した軍の所持ポイントが1,000ポイント未満の場合、倒された軍のポイントは0となり、リーダーを倒した軍は1,000ポイント入手できます。 | 50点 | シンボル像を破壊された軍は、その時点の得点の「25%」を失います。  シンボル像を破壊された軍が失った得点は、シンボル像に与えたダメージ量に応じて他の軍に分配されます。 |

| 魔法物質「ガーネット」 | 魔法物質「アクアマリン」 | 魔法物質「エメラルド」 | 魔法物質「トルマリン」 |
| --- | --- | --- | --- |
| 魔法物質「ガーネット」 | 魔法物質「アクアマリン」 | 魔法物質「エメラルド」 | 魔法物質「トルマリン」 |
| 1点 | 3点 | 5点 | 50点 |

#### リーダー戦演習の特典

騎士団演習中の活躍に応じて「獲得経験値」「獲得職業経験値」「獲得CP」を入手できます。

また、活躍に応じた様々な特別賞や、演習中の獲得ポイントや行動不能になった回数などを踏まえて最終的な特典が決まります。

※演習中に獲得ポイントを一定以上稼がないと入手できません。

![リーダー戦演習の特典](../../gamedata/system/_img/maneuvers/sys_man02_img_11.jpg)

| 特別賞の種類 | 内容 |
| --- | --- |
| ブービーofデッド | すべての参加者中、行動不能になった回数が2番目～11番目に多かったキャラクター10名に贈られます。 |
| ナイチンゲール賞 | すべての参加者中、他のキャラクターを回復した量がもっとも多かったキャラクター上位10名に贈られます。 |
| がんばリーダー賞 | すべての参加者中、リーダーであった時間がもっとも長かったキャラクター上位10名に贈られます。 |
| リーダーキラー賞 | すべての参加者中、もっとも多くのリーダーを倒したキャラクター上位10名に贈られます。 |
| 暴れん坊賞 | すべての参加者中、多くの敵軍キャラクターに対して最初にダメージを与えたキャラクター上位10名に贈られます。 |
| スーパーマン賞 | すべての参加者中、瞬間最大被ダメージがもっとも多かったキャラクター上位10名に贈られます。 |

1つの賞を獲得できるのは最大で10名までです。  
同じ順位に同率のキャラクターが複数いて10名の枠を超えてしまう場合、その順位以降は「ノーコンテスト（無効）」となります。
