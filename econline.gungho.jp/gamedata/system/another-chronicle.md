### アナザークロニクル　【あなざーくろにくる】

![](../../gamedata/system/_img/another-chronicle/sys_acn_img_01.jpg)

願望渦巻く本の世界。

そんな世界で物語を繰り返し続けるアナザー達の結末を変化させ、物語の紙片を手に入れる。

集まった紙片はページとなり、心象の見えなき者の手によってその形を取り戻し、その者に強大な力を貸すのである。

【ポイント】

- 「アナザークロニクル」を習得できるキャラクターは、BASELV100以上の三次職クロニクルジョブ、またはLV100パーツ入手「真の顔」イベントをクリアしたDEMである必要があります。
- 「アナザークロニクル」を使用するには「紙片」を揃える必要があります。

#### アナザークロニクルとは

エンシェントアークの「ＥＸキャプチャー」にてアナザーの「紙片」を集め、キャラクターに装着することで、短い間だけステータスを非常に大きく増大させ、専用のアナザースキルが使用できるようになる「アナザークロニクル」状態になることができます。

アナザークロニクルを習得するには、ＥＸキャプチャー「知りたがりの少女」をクリアし、その後「アナザーブック」の機能を開放するイベントを進行し、使用したいアナザーの紙片を一式揃える必要があります。

[＞＞ＥＸキャプチャーについてはこちら](../../gamedata/facility/ancientarc.html#pickup01)

イベントナビゲーション「アナザークロニクル入手」をクリアすることで「アナザーブック」の機能開放まで進行することができます。

![](../../gamedata/system/_img/another-chronicle/sys_acn_img_02.jpg)

#### アナザークロニクルの使用方法

アナザークロニクルを使用するには、「アナザーブック」に紙片をはめ込み「アナザーページ」を完成させ、装着する必要があります。

「アナザーページ」を装着すると、スキルウィンドウに「アナザー」のタブが出現します。
そこからスキル「アナザーブック開放」を使用することで、アナザークロニクル状態になることができます。

また「アナザーページ」は装着するだけで、「アナザーページ装着時」分のステータスが上昇します。

※装着しているアナザーページ毎に、ステータスの上昇内容やアナザースキルの内容は異なります。

![](../../gamedata/system/_img/another-chronicle/sys_acn_img_03.jpg)

##### アナザークロニクル中の効果

- キャラクターのステータスが大幅上昇
- 「アナザーブック開放」以外のアナザースキルが使用可能になります  
  ※アナザースキルはディレイアウトの効果の対象外になります。
- バフを解除するスキルの対象にならなくなる

##### アナザークロニクル中の解除条件

- 効果時間の経過
- 戦闘不能
- ログアウト
- マップ移動
- 「アナザークロニクル」中に憑依を行う

![ヒント](_img/sys_hin_cap_02.png)

- アナザーブックは「アナザーブック」の機能を開放するイベントをクリア後に表示される拡大した装備ウィンドウの本アイコンをクリックするか、メニューバーのキャラクター内の「アナザーブック」を選択することで開くことができます。
- アナザーページは、各ページに対応した紙片を全て当てはめた後、【作成】ボタンを押す事で完成することができます。
- ワールド対抗戦関連のMAPに移動しているキャラクターは「アナザーブック」を開く事ができません。ワールド対抗戦関連のMAPに移動前に装着していたアナザーページは装着したままのため、ワールド対抗戦でも対応するアナザースキルは使用可能になります。

#### アナザースキルの成長

「アナザーブック」の機能解放後から、モンスター討伐時やクエスト報酬などによって得られる基本経験値と同じ量の「ページ経験値」を獲得できるようになります。

ページ経験値はアナザースキルに振り分ける事でアナザースキルのスキルレベルを上昇させ、より強力にすることができます。

各アナザースキルの成長によって上昇する効果は以下の通りです。

| アイコン | 「アナザーブック開放」を成長 |
| --- | --- |
|  | - アナザークロニクル効果時間が上昇 - 「アナザーブック開放」使用時のステータス上昇量増加 |
| アイコン | その他のアクティブ・パッシブスキル |
|  | - 効果量の増加 - 効果範囲の変動 - 効果内容の変動 |

スキルレベルを上昇させるのに必要な経験値は、アナザースキルによって異なります。

![](../../gamedata/system/_img/another-chronicle/sys_acn_img_05.jpg)

#### アナザーページの複数回完成

「アナザーページ」は、ページ毎に５回まで重ねて完成することができます。

完成数が増えるほど、装着した際のステータス上昇量が増加したり、アナザースキルのスキルレベル上限が増加します。

各「アナザーページ」を２回目以降完成させるには、紙片を重ねた上で、完成回数に応じた「想いの万年筆」を消費する必要があります。

ページの完成数が１回増えるごとに、アナザースキルのスキルレベル上限が３ずつ上昇します。

![](../../gamedata/system/_img/another-chronicle/sys_acn_img_06.jpg)

#### 想いの万年筆と紙片の交換

想いの万年筆は、ＥＸキャプチャーで入手することもできますが、エンシェントアークロビーのＮＰＣ「紙芝居」に話しかける事で、一定量の紙片を想いの万年筆と交換することができます。

交換に使用する紙片は、同一のアナザーの紙片である必要がありますが、紙片の番号は問いません。

また「紙芝居」から交換した想いの万年筆は、交換したアナザーの紙片と同じ種類のアナザーページにしか使用できません。

想いの万年筆に必要な紙片の枚数は、想いの万年筆の種類によって異なります。

またNPC「紙芝居」をエンシェントアークロビーに出現させる為には、イベントナビゲーション「アナザークロニクル」内の「プロローグイベント～開幕～」をある程度進行している必要があります。

![](../../gamedata/system/_img/another-chronicle/sys_acn_img_07.jpg)

#### 拡張スキル

アナザースキルには、特定の職業スキルを習得している事で、より効果が増加したり、新たな効果が追加される「拡張スキル」があります。

拡張スキルには、デュアルジョブのスキルも効果対象に含まれ、またスキルレベルに応じて効果量も変化します。

同じアナザースキルに適用される、同じ効果の拡張スキルを所持している場合、効果が重複します。

![](../../gamedata/system/_img/another-chronicle/sys_acn_img_08.jpg)
