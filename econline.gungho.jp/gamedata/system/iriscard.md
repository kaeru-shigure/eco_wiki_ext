## システム

### イリスカードシステム【いりすかーどしすてむ】

![](_img/iriscard/sys_iri_img_01.jpg)

いろいろなものの「想い」がつまったカード。

攻撃属性値や防御属性値、「アビリティベクトル」と呼ばれる効果が付与されており、武器や防具に装着することで、冒険者をパワーアップすることができる。

【ポイント】

- 武器や防具に「スロット」を拡張し、「イリスカード」を装着することでさまざまな効果を得ることができます。
- イリスカードの購入、武具スロット追加や必要な材料の購入など、イリスカードに関連することは、「カードラボ」にて行うことが可能です。
- ある程度冒険をしたキャラクターで、アクロポリスシティ・ダウンタウンにいるNPC「壊し屋バキア」に話しかけることにより「カードラボ」へ案内してもらうことができます。
- イリスカードに付属している「アビリティベクトル」を組み合わせることで、「リリースアビリティ」という特殊な力を発揮することができます。

#### イリスカードの入手

「イリスカード」は、はじめから効果を持つイリスカードとして存在はしていません。  
カードラボで購入できる「ブランクイリスカード」や『ECO SHOP』から購入できる「ＥＸブランクイリスカード」を、 各地にいるNPC「夢見のハレルヤ」に渡すことで、特別な効果を持つ「イリスカード」へと変化します。  
また、中には「ブランクイリスカード」をドロップするモンスターや、「木箱」や「宝箱」から入手できるイリスカードが存在します。

イリスカードは[ＳＲ＞Ｒ＞ＵＣ＞Ｃ]の順番でレアリティが高いカードとなっています。

![](_img/iriscard/sys_iri_img_02.jpg)

##### イリスカードガチャ

各地にいるNPC「夢見のハレルヤ」に話しかけることで「イリスカードガチャ」が開きます。

![](_img/iriscard/sys_iri_img_18.jpg)

交換する為のブランクイリスカードを所持していなくても、交換できるイリスカードの一覧を閲覧することができます。  
またイリスカード一覧の各カードを右クリックすることで、各カードの詳細が表示されます。

#### イリスカードの保管

イリスカードは「カードホルダー」に収納することで他のアイテムと分けて保管することができます。  
また、「カードホルダー」は集めたイリスカードをお友達に自由に閲覧してもらうことが可能です。

カードホルダーは、「カードラボ」にいる「ジーニャ博士」から1ゲームアカウントにつき1個まで無料でもらうことができます。  
なくしてしまった場合は、10,000Goldで何度でも購入することができます。

![](_img/iriscard/sys_iri_img_03.jpg)

##### イリスカードを収納する

カードホルダーを開いて「イリスカードを収納する」を選ぶと、トレード画面が表示されます。収納したいカードをトレードウィンドウにドラッグ＆ドロップします。  
夢見のハレルヤから入手できる各カードにはナンバーが振られており、それぞれノーマルポケットの対応した箇所に収納されます。  
フリーポケットは、キャンペーンやオフラインイベント、ゲーム内の突発的なイベントなどで入手できるイリスカードが収納されます。フリーポケットに収納されるイリスカードにはナンバーがありませんので、カードホルダーに収納した順番に表示されます。

![](_img/iriscard/sys_iri_img_04.jpg)

##### イリスカードを取り出す

カードホルダーを開いて「イリスカードを取り出す」を選ぶと、カードホルダーの内容が一覧で表示されます。  
イリスカードをダブルクリックすることで、カードホルダーから取り出すことが可能です。

「カードホルダー」は飛空庭に家具として設置することで開くことができます。  
設置したら右クリックメニューから「カードホルダー」を選択して「使用」を選びましょう。

![](_img/iriscard/sys_iri_img_05.jpg)取り出したいイリスカードをダブルクリック

#### イリスカードを装着する

武器や防具、DEMのアームパーツやセットパーツに「スロット」を拡張し、イリスカードを装着することでさまざまな効果を得ることができます。  
スロットの拡張は、いくつかのアイテムとバックパッカー系のスキル、もしくはNPCに話しかけることでできます。

イリスカードを装着するには「スロット」の空いている武具が必要になります。  
また、各イリスカードはそれぞれ装着できる箇所が決められています。

![](_img/iriscard/sys_iri_img_06.jpg)

##### スロット拡張材料の入手

スロットを拡張するには、拡張材料アイテムが必要です。  
アイテムの装備可能レベルによって、必要なアイテムが異なります。また、拡張材料の素となるアイテムは、NPC「ジーニャ博士」より購入することが可能です。

拡張材料アイテムは、様々なアイテムを合成して作成する必要があります。

| アイコン | 名前 | 内容 |
| --- | --- | --- |
| 想いの結晶 | 想いの結晶 | 装備可能レベル低（LV1～LV30） |
| 想いの大結晶 | 想いの大結晶 | 装備可能レベル中（LV31～70） |
| 想いの真結晶 | 想いの真結晶 | 装備可能レベル高（LV71～LV110） |

装備可能レベル別拡張材料

![拡張材料合成一覧1](_img/iriscard/sys_iri_img_07.jpg)
![拡張材料合成一覧2](_img/iriscard/sys_iri_img_08.jpg)
![拡張材料合成一覧3](_img/iriscard/sys_iri_img_09.jpg)

##### 武器・防具、アームパーツ・セットパーツにスロットを追加する

拡張材料と装備アイテムを所持した状態で、カードラボまたはダウンタンにいる「灰のコルチカ」に話しかけるか、タタラベ系、ファーマー系またはレンジャー系の職業のキャラクターのスキル「アイテム精製」を使用して、スロットを拡張することができます。

スロットは胸アクセサリーには最大5スロット、上半身防具と右手装備、アームパーツとセットパーツには最大10スロットまでの拡張が可能です。  
※灰のコルチカでスロット拡張を行う場合、拡張材料に加えて、「装備可能レベル×1,000Gold」の手数料が必要です。

![](_img/iriscard/sys_iri_img_10.jpg)

【ご注意】

- ※スロットの拡張は、スロット数が多くなるほど、失敗する確率が高くなります。  
  拡張失敗時は拡張失敗のメッセージが表示され、装備アイテムと拡張素材アイテムが失われてしまいます。ご注意ください。

スキル「アイテム精製」を使用してスロット拡張をしたほうが成功確率が若干上がります。  
エクストラアイテムを使用することで、拡張の成功率を上げたり、装備アイテムの消失を防ぐことができます。  
またスロット拡張の際、エクストラアイテムの補助アイテムは、組み合わせによって二つ同時に使用することができる場合がございますのでご注意ください。  
さらに、お店で販売している一部の装備品には、スロットが初めから拡張されているものも存在します。

##### スロットにイリスカードを装着する

イリスカードを装着したいアイテムを右クリックし、アイテム情報ウィンドウを開いて「戦闘」タブをクリックします。  
右上にある「カード」ボタンを押すと、実際にカードを入れ替えできるウィンドウが開きます。  
※アイテムが故障した状態でイリスカードを取り出すことはできません。イリスカードを取り出す場合には、故障したアイテムを修理してください。

![](_img/iriscard/sys_iri_img_11.jpg)

スロットロックをしていない場合、戦闘中にカードが消失する可能性があります。  
一度スロットロックをしてしまうと、装着したイリスカードは通常の方法では取り出すことができません。  
充分に確認してからスロットロックを行うようにしてください。  
どうしてもイリスカードをはずしたい場合、スロットロックを解除するにはエクストラアイテム「スロットロック解除キー」が必要になります。

##### シンプルセッティングをする

「Set」ボタンを押すと、シンプルセッティングの設定画面が開きます。  
付与したい効果を選ぶと、所持しているカードの中からセッティングできるアビリティベクトルが表示されます。  
決定すると、自動的に選んだ効果を持つイリスカードがセットされます。  
※限定条件が設定されているカードは、シンプルセッティングの対象になりません。

![](_img/iriscard/sys_iri_img_11_02.jpg)

スロットロックをしていない場合、戦闘中にカードが消失する可能性があります。  
一度スロットロックをしてしまうと、装着したイリスカードは通常の方法では取り出すことができません。  
充分に確認してからスロットロックを行うようにしてください。  
どうしてもイリスカードをはずしたい場合、スロットロックを解除するにはエクストラアイテム「スロットロック解除キー」が必要になります。

#### イリスカードの効果

イリスカードには、攻撃属性値や防御属性値、「アビリティベクトル」と呼ばれる効果が付与されています。  
冒険で訪れる地によって、イリスカードを効果的に使い分けましょう。

![](_img/iriscard/sys_iri_img_12.jpg)

##### アビリティベクトルとリリースアビリティを覚えよう

イリスカードは、それぞれ「アビリティベクトル」という能力を持っています。

アビリティベクトルにはイリスカードが持つ「力」がどの程度強くこもっているのかを表すベクトル値があり、ある一定の値を超えると、「リリースアビリティ」と呼ばれる様々な効果が発動し、キャラクターをパワーアップさせることができます。

![](_img/iriscard/sys_iri_img_15.jpg)

同じアビリティベクトルを持ったイリスカードを複数枚使った場合、各カードのベクトル値は加算されます。このベクトル値が高くなると、発動するリリースアビリティのLVも高くなります。

同じ種類の能力を持つリリースアビリティは、同時に適用されません。使っているカードの効果の中で、最も効果の高いもののみが適用されます。

| アビリティベクトルのLv | アビリティベクトルの合計値 |
| --- | --- |
| Lv1 | 1 |
| Lv2 | 30 |
| Lv3 | 80 |
| Lv4 | 150 |
| Lv5 | 250 |
| Lv6 | 370 |
| Lv7 | 510 |
| Lv8 | 660 |
| Lv9 | 820 |
| Lv10 | 999 |

##### 属性値補正を覚えよう

属性値補正は、イリスカードを装着する部位によって攻撃属性か防御属性に分けられます。

上半身装備(上半身を含む装備)に装着できるイリスカードの場合、属性値は「防御属性値」となり、右手装備(右手を含む装備)の場合は「攻撃属性値」となります。

![表示されるアイコンと意味する属性](_img/iriscard/sys_iri_img_14.jpg)

「カード情報ウィンドウ」の「効果」タブで、イリスカードが持つ属性値補正の詳細を確認することができます。

#### スロット拡張で経験値を獲得

メインジョブがバックパッカー系の職業のキャラクターは、スキル「アイテム精製」を使用してスロット拡張を行うと、基本経験値と職業経験値を獲得することが出来ます。

- 獲得できる経験値量はキャラクターのBaseLV、JobLV、空いているスロット数に応じて決まります。
- メインジョブがバックパッカー系でない場合、経験値を獲得できません。
- スロット拡張に失敗した場合は、経験値を獲得できません。

![スロット拡張で経験値を獲得](_img/iriscard/sys_iri_img_17.jpg)

##### 他のプレイヤーにスキルを使用してもらった場合

バックパッカー系のキャラクターで、他のキャラクターにスキルを使用してもらい、  
スロット拡張を行った場合でも経験値を入手できます。

- 他人にスキルを使用しても、自分のキャラクターは経験値を獲得できません。
- NPCにスキルを使用してもらった場合は経験値を獲得できません。

##### スロット拡張の適正LVについて

スロット拡張には拡張する武具の装備LV毎に適正LVが設けられており、自分のキャラクターと装備LVが離れたアイテムをスロット拡張しても、得られる経験値が本来のものよりも少なくなります。

- スロット拡張の適正LVはBaseLVを基準に判断されます。
