## システム

### 武具融合　【ぶぐゆうごう】

![](_img/fusion/sys_fus_img_01.jpg)

2つの異なる装備アイテムの「見た目」と「能力」を1つに融合するシステム。

「グラフィックは気に入っているが性能は低い装備」と、「性能は高いがグラフィックは好みではない装備」を融合させることで、「性能が高くグラフィックも気に入った装備」に生まれ変わらせることができる。

【ポイント】

- 「自分の好みに合ったグラフィックと、キャラクターのレベルに見合った性能」を持つ装備アイテムにすることができます。
- 武具融合は何度でも行うことができます。
- 武具融合は、フシギ団の砦・聖女のシマのNPC「ヨーコ」、ウェストフォートの裁縫屋、エル・シエルの裁縫屋に話しかけるか、「ハーヴェスト」のスキル「武具融合」を使用することで行えます。  
  「ハーヴェスト」のスキル「武具融合分解」を使用すると、融合した武具を2つのアイテムに戻すことができます。

#### 武具融合の条件

「武具融合するためには、下記の条件を満たした2つの装備アイテムが必要になります。  
同じ種別タイプの装備アイテムである。

※装備箇所・モーションが別でも大丈夫です。  
銃系統の武器以外の場合は攻撃の「射程」が同一である必要があります。また、エフェクト装備の融合が可能になりました。

「見た目に使用する装備」と「能力に使用する装備」の装備可能条件（LV・職業・種族・性別・上位転生など）の一部が合致している。

![](_img/fusion/sys_fus_img_02.jpg)

上記、同じ種別タイプの装備アイテムという融合条件の例外的組み合わせとして、以下の表の組み合わせがあります。

| 見た目アイテム |  | 能力アイテム |
| --- | --- | --- |
| 顔装飾品 | + | フルフェイス |
| 頭部装飾品 | + | 頭部防具 |
| 頭部装飾品 | + | フルフェイス |
| 頭部防具 | + | 頭部装飾品 |
| 頭部防具 | + | フルフェイス |
| 胸部装飾品 | + | シンボル |
| 靴 | + | ブーツ |
| 靴 | + | ハーフブーツ |
| ブーツ | + | ハーフブーツ |

下記のアイテムは武具融合に使用できません。

|  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- |
| 投擲武器 | 各種の矢 | 各種の弾 | ペット装備 | イベントアイテム | 故障している装備アイテム |

アイテムを右クリックすると、「アイテム情報」が開けます。融合したいアイテムの詳細を確認しておこう。

#### 融合をするには

融合したいアイテムを所持した状態で、フシギ団の砦・聖女のシマのNPC「ヨーコ」、ウェストフォートの裁縫屋、エル・シエルの裁縫屋に話しかけるか、「ハーヴェスト」のスキル「武具融合」を使用することで行えます。

NPCより武具融合を行う場合、「手数料」が必要になります。

![](_img/fusion/sys_fus_img_03.jpg)

##### 武具融合が成功した場合

武具融合が成功すると、「見た目」アイテムは「能力」アイテムに融合されます。  
そのため、「見た目」アイテム自体はなくなってしまいます。   
武具融合に成功した装備のアイテムアイコンは、「能力」アイテムの方のアイコンが☆印付きでアイテムウィンドウに表示されます。  
※「見た目」アイテムの強化値・スロット・装着していたイリスカードは消滅しますのでご注意ください。

![](_img/fusion/sys_fus_img_04.gif)

##### 武具融合が失敗した場合

武具融合に失敗しても「見た目」アイテム、「能力」アイテムはどちらも失われません。  
また、武具融合で使った手数料は失われます。

「おしゃれな糸」を使用すると、武具融合の成功確率を上げることができます。

#### 融合を解除するには

融合した装備アイテムは、「ハーヴェスト」のスキル「武具融合分解」を使用することで、2つのアイテムに戻すことができます。

![](_img/fusion/sys_fus_img_05.jpg)
