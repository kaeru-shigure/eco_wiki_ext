## システム

### DEM・DEMIC 【でむ・でみっく】

![DEM・DEMIC](_img/dem/sys_dem_img_01.jpg)

己を構成するパーツやチップを組み替えることで、様々な状況に立ち向かえる種族。

他者との繋がりによって心の力を得ることで成長するという。

【ポイント】

- DEMは状況に応じて様々な「フォーム」に変形することができます。
- 「DEMパーツ」をカスタマイズしたり「DEMチップ」を組み合わせる「DEMIC」を行うことでステータスやスキルを変更できます。
- EP（Engage Point）を使用することで、CL（コストリミット）を増やしてLVを上げたり、カスタマイズや「DEMIC」を行うことができます。
- パーツのカスタマイズや「DEMIC」などは、「カスタマイズ部屋」またはエクストラアイテムを使用することで行うことができます。
- アクロポリスシティ・ダウンタウンにいるNPC「ラボの案内人」またはウェストフォートの「スナブ」に話しかけると、「カスタマイズ部屋」に行くことができます。

#### フォームチェンジ

キャラクターの右クリックメニューより、通常時の「ノーマルフォーム」と戦闘に特化した「マシナフォーム」に切り替えることができます。

##### ノーマルフォーム

![ノーマルフォーム](_img/dem/sys_dem_img_02.jpg)

種族専用装備以外であれば、武器や防具、ペットなど、なんでも装備が可能です。

しかし、DEMICに装着したチップのステータスやスキルが一切反映されません。

##### マシナフォーム

![マシナフォーム](_img/dem/sys_dem_img_03.jpg)

装着した「DEMパーツ」の姿となり、DEMICに装着したチップのステータスやスキルが反映されます。

しかし、ノーマルフォーム時に装備している武器や防具の性能は反映されず、一部のペット以外はペットを装備することができません。

DEMは一部のマシナフォームを除き、憑依をすることができません。  
また、マリオネットを使用することができません。

#### DEMのLV・ステータスについて

DEMは他の3種族の成長方法とは異なり、経験値を獲得することでLVを上げることはできず、[EP（Engage Point）](../../../econline.gungho.jp/gamedata/system/heartskill.html)を消費して、CL（コストリミット）を増やすことでLVが上がります。

ただし、LVが上がるだけではステータスは変化しません。

DEM専用のパーツによる『カスタマイズ』やチップを使った『DEMIC』により、ステータスが上昇したり、スキルが使えるようなります。

DEMキャラクターで特定のイベントを行うと、「EPバッテリー」を得ることがあります。EPバッテリーがたまった場合、カスタマイズ部屋のジーニャ博士かスナブに話しかけると、たまったEPバッテリーの値を自分のEPとしてチャージしてもらうことが可能です。

![DEMのLV・ステータスについて](_img/dem/sys_dem_img_04.jpg)

#### カスタマイズ・DEMIC

「カスタマイズ」・「DEMIC」は、アクロポリスシティ・ダウンタウンにいるNPC「ラボの案内人」、またはウェストフォートのNPC「スナブ」に話しかけると行ける「カスタマイズ部屋」で行うことができます。

エクストラアイテム「どこでもカスタマイズ（３０日）」を使用すると、どこにいても「カスタマイズ」・「DEMIC」を行うことができます。

![カスタマイズ・DEMIC](_img/dem/sys_dem_img_05.jpg)

##### パーツカスタマイズ

DEMに装着しているパーツの付け替えをすることができます。  
装着するパーツによってステータスやスキルが変わります。

パーツはカスタマイズ部屋のNPC「クォーク博士」またはNPC「スナブ」や、「ECO SHOP」から購入することができます。

![パーツカスタマイズ](_img/dem/sys_dem_img_06.jpg)

#### DEMICの遊び方

「DEMICパネル」にチップを設置することでステータスやスキルを変更することができます。  
DEMICを行うにはEPを「５」消費します。

![DEMICの遊び方](_img/dem/sys_dem_img_07.jpg)

| No. | 名称 | 画面の説明 |
| --- | --- | --- |
| 1 | DEMICパネル | このパネルに各種チップを装着すると、ステータスやスキルの効果を得ることができます。  パネル内の明るく表示されている箇所に、各種チップを装着できます。    パネルは、EPを消費してコストリミット（CL）を増加させると拡張することができます。 |
| 2 | DEMICパネルページ切り替え | DEMICパネルは通常、最大で6ページまで拡張することができます。  各種チップは、ページごとに入れ替えができます。    また、エクストラアイテムの「コストリミット追加α（30日）」を所持していると「α」ページを、「コストリミット追加β（30日）」を所持していると「β」ページを開くことができます。  追加されるページは、α、βともにコストリミット81マスです。最大で162マスが拡張できます。 |
| 3 | エンゲージタスク | DEMICの右上に２つあり、DEMICパネル内にドラッグ＆ドロップすることで任意の場所に設置する事ができます。  エンゲージタスクに隣接して配置した各種チップは、通常の倍の効果を得る事ができます。  エンゲージタスクの効果を得ているチップは点滅して表示されます。  エンゲージタスクをDEMICパネルに設置するにはEPを「１」消費します。 |
| 4 | 所持しているチップの一覧 | 現在所持しているチップがカテゴリ別で表示されます。  このウィンドウから、DEMICパネル内にドラッグ＆ドロップすることで、各種チップを装着することができます。 |
| 5 | スキル・ステータス確認ウィンドウ | 現在のステータスと所持スキル確認ウィンドウです。  これらのステータスやスキルは、マシナフォームになったときに反映されます。 |
| 6 | スキルチップ重ねがけ詳細ウィンドウ | 現在のスキルレベルが確認できます。ゲージが貯まる毎にスキルレベルが1上昇します。 |

#### 各種チップについて

DEMICパネルに設置できる各種チップは、大きく分けて3種類あります。

|  |  |  |
| --- | --- | --- |
| ステータスチップ | ステータスチップ | 各ステータスごとの6種類のチップが存在し、セットすることで各種ステータスが上がります。 |
| スキルチップ | スキルチップ | 様々なスキルチップが存在し、セットすることで各種スキルを使用できるようになります。  複数セットすることでスキルのLVが上がります。  また、DEMキャラクターのLvが100になると同じスキルチップを重ねておく事ができるようになります。 |
| リミテーションチップ | リミテーションチップ | さまざまな戦闘スタイルに合わせて、補正効果を得ることができるチップ。 リミテーションチップはDEMICパネル内に1つしか装着できません。 |

DEMチップはカスタマイズ部屋のNPC「クォーク博士」またはNPC「スナブ」から経験値と引き換えに入手したり、「ECO SHOP」から購入することができます。  
「ECO SHOP」から入手できるDEMチップは、占有マスの少ないものになります。

#### スキルチップ重ねがけについて

LV100以上のDEMは、「同一のスキルチップ」を複数枚重ねて設置する事ができます。

すでに設置されているスキルチップに対し、同一のスキルチップをドラッグ＆ドロップすることにより、スキルチップを重ねることができます。

スキルチップを重ねることでDEMIC画面左下のゲージが貯まり、スキルLVが上昇します。

![キルチップ重ねがけについて](_img/dem/sys_dem_img_08.jpg)
