## システム

### パーティー　【ぱーてぃー】

![](_img/party/sys_par_img_01.jpg)

他のキャラクター達と組んで冒険することで様々な局面に立ち向かうことができる。

親しい友は冒険と共に。

【ポイント】

- 他のキャラクターを交え、最大8人で一緒に冒険に出ることができます。
- パーティーメンバーが倒したモンスターの経験値やドロップしたアイテムを分配することができます。

#### パーティーのメリット

パーティーを組んだメンバーで倒したモンスターの経験値を分配することができ、
パーティーメンバーの数により獲得経験値にボーナスが入ります。  
また、モンスターがドロップしたアイテムを分配することができます。

##### 経験値

攻撃したパーティーメンバー数が多いほどボーナス経験値は高くなります。

パーティーメンバーが同じパーティーのメンバーに憑依していた場合、
憑依者はパーティー経験値と同時に憑依の経験値も得ることができます。

各パーティーメンバーに分配される経験値は、キャラクターのレベルによって異なり、
キャラクター同士のレベル差があればあるほどパーティーメンバーが獲得できる経験値が減ります。

モンスターを倒したパーティーメンバーから遠く離れていたり別のMAPにいる場合や
ログアウトしている時は、経験値分配対象とはならず、ボーナス経験値算出人数にも含まれません。  
また、憑依者の攻撃は宿主の攻撃とみなされ、ボーナス経験値算出人数には含まれません。

![](_img/party/sys_par_img_02.jpg)

##### アイテムの分配

モンスターがドロップしたアイテムの分配方法は2種類のモードから選べます。

アイテム分配方法はパーティーリーダーのみが変更できます。

![](_img/party/sys_par_img_03.jpg)

###### 通常モード(パーティーメンバーフリー取得)

モンスターがドロップしたアイテムをパーティー内の誰もが最初から取得できるモードです。

|  |  |
| --- | --- |
| パーティーメンバーが取得できる時間 | ～60秒 |
| 誰でも取得できる時間 | 61～120秒 |
| 消滅時間 | 121～180秒 |

###### ランダムモード(パーティー分配取得)

モンスターがドロップしたアイテムはパーティーメンバーの誰かが優先取得できるモードです。

優先取得できるパーティー内順位はランダムになります。

優先権が設定されたパーティーメンバーには、優先権があることを示すエフェクトが表示されます。

※優先権のない他のメンバーには表示されません

モンスターを倒したパーティーメンバーから遠く離れていたり別のMAPにいる場合や
ログアウトしている時は、経験値分配対象とはならず、ボーナス経験値算出人数にも含まれません。  
また、憑依者の攻撃は宿主の攻撃とみなされ、ボーナス経験値算出人数には含まれません。

|  |  |
| --- | --- |
| 優先権があるメンバーだけが取得できる時間 | ～30秒 |
| パーティーメンバーが取得できる時間 | 31～60秒 |
| 誰でも取得できる時間 | 61～120秒 |
| 消滅時間 | 121～180秒 |

![](_img/party/sys_par_img_04.jpg)

#### パーティーを組む

パーティー作成者がリーダーとなり、パーティーメンバーを最大8人まで増やすことができます。

パーティーに誘いたいキャラクターを右クリックし、「パーティーに誘う」を選び、 誘われたキャラクターが「OK」を押すと、パーティーへ参加できます。

また、フレンドリストのフレンドタブやリングタブの中からでも、同様にパーティへ誘うことが可能です。

パーティーメンバーの情報は、「フレンドリスト」の「パーティー」タブで確認できます。

パーティーから脱退、もしくはパーティーが解散しない限り、ログアウト後もパーティー状態が続きます。

![](_img/party/sys_par_img_05.jpg)

#### パーティーリーダーの権限

パーティーリーダーは、様々な権限を持つことができます。

「パーティードロップ分配の変更」は「フレンドリスト」の「パーティー」タブから行うことができます。

自分のキャラクターを右クリックすると、
「 パーティー名変更」 と「パーティーを解散する」を選ぶことができます。

パーティーを脱退させたいメンバーがいる場合、
脱退させたいメンバーの名前を右クリックして「パーティーを脱退させる」を選択すれば、
自分以外のパーティーメンバーを強制的に脱退させることができます。

パーティー名は全角8文字・半角16文字まで入力が可能です。

![](_img/party/sys_par_img_06.jpg)

#### パーティーからの脱退

パーティー状態のときに自分のキャラクターを右クリックし、
「パーティーから抜ける」を選択すると、パーティーを脱退することができます。

パーティーを組んだ人と仲良くなって、また一緒に冒険にでよう。

![](_img/party/sys_par_img_07.jpg)
