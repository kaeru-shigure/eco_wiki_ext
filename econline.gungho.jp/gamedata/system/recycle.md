## システム

### リサイクルシステム【りさいくるしすてむ】

![](_img/recycle/sys_rec_img_01.jpg)

アクロニア大陸の各地に存在する謎の装置。

アイテムからエネルギーを取り入れることで、冒険者個人や世界に様々な恩恵を与えるという。

【ポイント】

- 各地に存在する「リサイクル装置」に一部のアイテムを納品すると、「ワールドゲージ」と「リサイクルポイント」が貯まります。
- ワールド共通の「ワールドゲージ」が100%に達すると、2週間そのワールドに様々な特典が発生します。
- リサイクル装置で、「リサイクルポイント」を特別な自販機で使える「ＥＣＯメダル」に交換できます。

#### リサイクル装置とは

リサイクル装置にさまざまなアイテムを入れる事でエネルギーを貯め、ワールド単位で色々な特典を発生させることができます。

また、一部のアイテムを納品すると、特殊な自販機で使用できる「ECOメダル」に交換することができる「リサイクルポイント」を入手することができます。

「リサイクルポイント」は、キャラクターごとに貯まります。

![](_img/recycle/sys_rec_img_02.jpg)

| リサイクル装置のあるマップ | | | |
| --- | --- | --- | --- |
| ダウンタウン | タイニーアイランド | タイタニアビーチ | ウェストフォート |
| エル・シエル（下層） | トンカ島 | ファーイーストシティ | モーグシティ |
| ノーザンプロムナード | アイアンシティ上層階 | イストー岬 | 軍艦島 |
| ノーザリン岬 | サウスリン岬 |  | |

#### リサイクル装置を使う

リサイクル装置をクリックして、納品ウィンドウに納品したいアイテムを入れると、ワールド共通の「ワールドゲージ」が貯まります。

また、もてなしタイニーから貰える『おもてなしアイテム』やゲーム内の貴重なアイテムの中には、リサイクル装置に入れた時に「リサイクルポイント」をもらえる物があります。

リサイクルポイントはキャラクター単位で保存されます。また、もらえるリサイクルポイント数はアイテムによって異なり、リサイクルポイントがもらえないアイテムもあります。  
納品したアイテムをもとに戻すことはできません。

![](_img/recycle/sys_rec_img_03.jpg)

##### リサイクルできないアイテム

- ゲーム内のNPCで販売されているアイテム
- イベントアイテム
- （クエスト）と記載されたアイテム
- ECOタウンやエレキテルラボ、都市攻防戦などのポイント交換で得られるアイテム
- BOXやパック、箱などのアイテム
- 当月に販売されているECOくじ
- ECO SHOPで販売中のエクストラアイテム

【ご注意】

- ※リサイクルできないアイテムには一部例外があります。
- ※BOXやパック、箱などのアイテムは、一部例外を除き開封した中身をリサイクルする事は可能です。
- ※ECOくじは、販売より一定期間経過すると納品することができるようになります。

#### ワールドゲージの特典

ワールドゲージが100％になると、2週間の期間、ワールド特典が発生します。

その際、リサイクル装置が赤く点灯し、ワールド全体にアナウンスが流れます。

ワールド特典が発生している間は、リサイクル装置にアイテムを納品しても、ワールドゲージを貯めることはできません。  
ワールド特典の終了と共に、ワールドゲージは0％に戻ります。

![](_img/recycle/sys_rec_img_04.jpg)

##### ワールド特典の効果

ワールド特典の効果は、イベントなどによって不定期に変更されます。

| リサイクル装置で発生する特典 | |
| --- | --- |
| 基本経験値・職業経験値1.5倍 | カードアセンブル成功率上昇 |
| 武具強化成功確率上昇 | カードスロット拡張成功率上昇 |

#### リサイクルポイントの特典

リサイクル装置をクリックして『「ＥＣＯメダル」と交換』を選ぶと、「2,000 リサイクルポイント」を消費する事で、『ＥＣＯメダル』を入手する事ができます。

リサイクル装置をクリックして行くことができる「秘密の場所」にはＥＣＯメダルを使用する特殊な自販機があり、過去のECOくじアイテムを入手することができます。

ＥＣＯメダルは、「もてなしタイニー」から入手することもできます。
