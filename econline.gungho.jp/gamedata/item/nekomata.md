### 背負い魔

#### ネコマタ

ネコマタには「戦闘型」、「支援型」、「特殊能力型」の3タイプがあり、キャラクターに背負われながら成長し、その能力を強化していきます。

##### ネコマタ(桃)

![](_img/ite_sdv_nmt_01.jpg)

|  |  |  |
| --- | --- | --- |
| 種別 | 「支援能力」と「特殊能力」を有する複合タイプ | |
| 装備条件 | キャラクターの性別や種族、職業、レベルなどの制限はありません。 | |
| 支援能力 | 回復支援 | 一定の確率で、回復魔法をかけます。 |
| 戦闘支援 | 一定の確率で、戦闘を支援する魔法をかけます。 |
| 特殊能力 | 自律行動 | プレイヤーに話しかけてきます。 |
| 所持量増加 | キャラクターの最大ペイロードと最大キャパシティーが増加します。 |

- ![](_img/ite_sdv_nmt_01_2.jpg)

##### ネコマタ（新緑）

![](_img/ite_sdv_nmt_02.jpg)

|  |  |  |
| --- | --- | --- |
| 種別 | 「特殊能力」を有するタイプ （「覚醒」すると、支援能力も有します） | |
| 装備条件 | キャラクターの性別や種族、職業、レベルなどの制限はありません。 | |
| 支援能力 ※「覚醒」状態時 | 回復支援 | 一定の確率で、回復魔法をかけます。 |
| 戦闘支援 | 一定の確率で、戦闘を支援する魔法をかけます。 |
| ※あるイベントによって「覚醒」させることで、上記のような支援能力を発揮します。 | |
| 特殊能力 | 自律行動 | プレイヤーに話しかけてきます。 |
| 所持量増加 | キャラクターの最大ペイロードと最大キャパシティーが増加します。 |
