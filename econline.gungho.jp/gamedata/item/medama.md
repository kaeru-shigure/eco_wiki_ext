### 背負い魔

#### メダマー

背負うことでキャラクターを回復や戦闘支援を行ってくれる背負い魔です。

##### メダマーキーパー

![メダマーキーパー](_img/ite_sdv_mdm_01.jpg)

|  |  |  |
| --- | --- | --- |
| 種別 | 「支援能力」と「特殊能力」を有する複合タイプ | |
| 装備条件 | ハイエミル  ※入手するためには、エミル種族で特定の条件を満たした上で上位転生する必要があります。 | |
| 支援能力 | 回復支援 | 一定の確率で、回復スキルを使用します。 |
| 戦闘支援 | 一定の確率で、戦闘を支援するスキルを使用します。  主に防御スキルを使います。 |
| 特殊能力 | ※この背負い魔は成長しません。 | |
