### 背負い魔

#### ブースト

背負う（装備する）ことでキャラクターの移動速度を一時的に向上させることができます。

##### ブースト

![](_img/ite_sdv_bst_01.jpg)

|  |  |
| --- | --- |
| 種別 | 「特殊能力」に特化したタイプ |
| 装備条件 | エミルのみ装備することができます。 |
| 特殊能力 | 装備すると、一時的にキャラクターの移動速度を上昇させるアクティブスキル「点火」を使用できるようになります。  ※この背負い魔は成長しません。 |
|  | |
