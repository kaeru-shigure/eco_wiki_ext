### 背負い魔

#### オートマタ

オートマタには「戦闘型」、「支援型」、「特殊能力型」の3タイプがあり、DEMキャラクター専用の背負い魔です。

##### オートマタ（アメジスト）

![オートマタ（アメジスト）](_img/ite_sdv_amt_01.jpg)

|  |  |  |
| --- | --- | --- |
| 種別 | 「支援能力」と「特殊能力」を有する複合タイプ | |
| 装備条件 | キャラクターの性別や種族、職業、レベルなどの制限はありません。  ※入手できるのはDEMのキャラクターのみです。 | |
| 支援能力 | 回復支援 | 一定の確率で、回復スキルを使用します。 |
| 戦闘支援 | 一定の確率で、戦闘を支援するスキルを使用します。 |
| 特殊能力 | 自律行動 | プレイヤーに話しかけてきます。 |
| 所持量増加 | キャラクターの最大ペイロードと最大キャパシティーが増加します。 |
| ディメンションパーツ化  ※DEM用 | DEMキャラクターが特定の条件を満たすことで、DEMパーツにアイテムチェンジを行うことができます。 |
