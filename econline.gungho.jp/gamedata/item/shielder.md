### 背負い魔

#### シールダー

背負う（装備する）と、シールダーから伸びた腕が、キャラクターを敵の攻撃から守ってくれることがあります。

##### シールダー

![](_img/ite_sdv_sld_01.jpg)

|  |  |
| --- | --- |
| 種別 | 「特殊能力」に特化したタイプ |
| 装備条件 | エミルのみ装備することができます。 |
| 特殊能力 | 装備していると一定確率で物理攻撃、スキル攻撃を回避できます。  ※この背負い魔は成長しません。 |
