## クライアントのダウンロード

会員登録がすんだらゲームクライアント（ゲーム本体）をダウンロードしましょう。  
また、ご利用のパソコンがエミル・クロニクル・オンラインの動作環境を満たしているか、あわせて確認ください。

ECOは、パソコン性能が比較的低くても遊べる、はじめてプレイされるかたにもやさしい、オンラインゲームです。

![](_img/download/down_img_01.jpg)

##### ECOで遊ぶ

[![クライアントをインストールして遊ぶ](_img/download/dow_dl_f1.png "クライアントをインストールして遊ぶ")](http://cldownload.gungho.jp/eco/ECO_SETUP.exe)

#### 動作環境

| OS | 日本語版 Microsoft Windows 7 32bit/64bit 版 日本語版 Microsoft Windows 8.1 32bit/64bit版 日本語版 Microsoft Windows 10 32bit/64bit 版 |
| CPU | Pentium4 3GHz 以上 |
| メモリ | 2GB 以上 |
| グラフィックスカード | 128MB以上のVRAMを有する3Dアクセラレーターチップ搭載ボード |
| ハードディスク | 10GB以上 |
| ブラウザ（32bit版のみ） | OSにより動作環境が異なります。  ・Winsows7 SP1:Internet Explorer11  ・Windows8.1:Internet Explorer11  ・Windows10:Internet Explorer11 |
| その他 | DirectX 8.1 以上(※1) ADSL 1Ｍbps以上のインターネット接続が常時可能な環境 日本語環境 MS-IME Windows Media Player 9.0以上必須(※2) |

※1  
DirectXとは、Microsoft社がWindowsシリーズのマルチメディア機能を強化するために提供しているアプリケーションです。 DirectXを使えば、直接ハードウェアを制御することが可能になるため、ゲーム機などの専用ハードウェアに負けない高度なマルチメディア処理をパソコン上で実現することができます。DirectXは3次元グラフィックスや音声を多用したゲームの開発に利用されることが多く、ECOもDirectX 8.1を用いて動きます。  
[DirectXのダウンロード](http://www.microsoft.com/japan/directx/default.mspx)

※2  
Windows Media Playerとは、Microsoft社が提供している動画や音声の再生を行うアプリケーションで、Windowsには標準でインストールされています。 ECOではゲーム内にてWindows Media Player 9.0以上の機能を使用して動画の再生などを行います。

■「稼動・推奨スペック」について

運営チームでは、「エミル・クロニクル・オンライン」をご利用いただくために、動作確認を行った上で「稼動・推奨スペック」を掲載しております。  
しかしながら、ユーザー様のご利用環境により、正常にプレイできない場合がございます。予めご了承ください。"

■「稼動・推奨スペック」を満たしていないパソコンについて

運営チームでは、動作確認を行っていない「稼動・推奨スペック」以外のパソコンをご利用の場合、サポート対象外とさせていただきます。 予めご了承ください。

■「自作」「ショップブランド」パソコンについて

「自作」「ショップブランド」パソコンにつきましては、ユーザー様ごとに環境が異なるため、動作を確認することはできません。  
運営チームでは、上記に該当するパソコンをご利用の場合、サポート対象外とさせていただきます。予めご了承ください。  
なお、運営チームにて動作検証を行い、動作確認を行った「エミル・クロニクル・オンライン推奨パソコン」については、サポート  
の対象とさせていただきます。

■「メーカー製」パソコンの部品の交換、増設、改造について

お使いのパソコンが「稼動・推奨スペック」を満たしていない場合、部品の交換、増設、改造などを行うことにより、「稼動・推奨スペック」と同等にプレイできる場合があります。  
しかしながら、メーカー側では、上記に該当するパソコンの動作を保証することができず、サポートを受けることができない場合がございます。  
運営チームにおいても、上記に該当するパソコンの動作を確認することができませんので、サポート対象外とさせていただきます。 予めご了承ください。
