下記①の不具合に伴う対応として、本日の定期メンテナンス時に、  
対象の皆様へ以下の対応を実施いたしました。

**▼対象**  
7月28日（木）定期メンテナンス時にリセット対象となった  
各種パイレーツマスケットを所持していたキャラクター

**▼対応**  
各種パイレーツマスケットに対して使用したエクストラアイテム等を  
ゲーム内の「ギフトシステム」で返却いたしました。

対象キャラクターを所持されている皆様におかれましては、  
ゲーム内の「ギフト」より、お受け取りくださいますようお願いいたします。

※ギフトシステムについては、一度に受信できる数が10通までとなっております。  
アイテムをご確認いただけない場合は、お手数ではございますが以下手順で  
ご確認くださいますようお願いいたします。

**≪アイテム確認手順≫**  
1.ギフトウィンドウで現在受信しているアイテムを受け取る  
2.一旦ゲームからログアウト  
3.再度ゲームへログインしギフトウィンドウを開く

[2016年7月28日　更新]  
下記①、③～⑥の不具合について、本日の定期メンテナンスで修正しましたことをご報告いたします。

7月21日（木）定期メンテナンス以降、以下の不具合を確認しております。

###### 不具合詳細

①各種「パイレーツマスケット」の性能開放を行った際、本来増減するはずのステータスではなく、  
異なったステータスが増減してしまっている。

**●「パイレーツマスケット（金）」「パイレーツマスケット（銀）」**

|  |  |
| --- | --- |
| 本来増減するステータス | 現在増減しているステータス |
| STR | MAG |
| INT | AGI |
| 物理攻撃力（突） | 魔法攻撃力 |
| 近距離命中力 | 魔法命中力 |
| クリティカル力 | クリティカル回避 |

**●「パイレーツマスケット（２丁・金）」「パイレーツマスケット（２丁・銀）」  
「パイレーツマスケット（２丁・MIX）」**

|  |  |
| --- | --- |
| 本来増減するステータス | 現在増減しているステータス |
| INT | DEX |

②各種「サーフトップス♀」「サーフパンツ」のアイテム情報ウィンドウに「○○状態を解く」という  
実際の仕様とは異なった情報が表示されている。

③チュートリアルイベントの「イリス博士」の台詞にて、会話ウィンドウ内に何も表示されない箇所がある。

④各種イベントにおいて会話ウィンドウ内に不適切な文字列が表示される。

⑤まいにちダンジョンの経験値ダンジョンにおいて、「ヌシ」の台詞が改修前の内容になっており、  
実際にもらえる経験値と倍率が異なっている。

⑥各種「ベーカリートップス」のグラフィックが特定モーションを行った際、正常に表示されない。

###### 不具合の修正について

**①の不具合について**

7月28日（木）定期メンテナンスにて不具合の修正と以下のアイテムに対して行われている  
性能開放・武具強化・潜在武具強化・スロット拡張およびスロットにセットしていた  
イリスカード・耐久度および最大耐久度の情報をリセットいたします。

後日、性能開放・武具強化・潜在武具強化・スロット拡張の際に使用したエクストラアイテムと  
耐久度および最大耐久度を回復させたエクストラアイテム、セットしていたイリスカードを返却させていただく予定です。

念のため、7月28日（木）定期メンテナンス修正までの間、  
以下のアイテムで性能開放・スロット拡張・武具強化・潜在武具強化・耐久度  
および最大耐久度の回復を行わないようお願いいたします。

また、該当アイテムにすでにイリスカードを挿していた場合は修正までの間、  
外していただけますようお願いいたします。

**▼対象アイテム**・パイレーツマスケット（金）  
・パイレーツマスケット（銀）  
・パイレーツマスケット（２丁・金）  
・パイレーツマスケット（２丁・銀）  
・パイレーツマスケット（２丁・MIX）

**②の不具合について**後日修正を行います。  
修正の際には追って公式サイトにてご案内させていただきます。

**③～⑥の不具合について**7月28日（木）定期メンテナンスにて修正を行います。

ユーザーの皆様にはご迷惑をおかけしておりますことをお詫び申し上げます。
