### ゲームサービス終了のお知らせ重要なお知らせ

![更新](../../../../econline.gungho.jp/news/_img/icon_update.gif)
最終更新日：2017/05/15
11:00
　掲載日：2017/05/15 10:00

ECO運営チームです。

2014年7月よりお楽しみいただいてまいりましたエミル・クロニクル・オンラインでございますが、  
このたび、2017年7月2７日（木）をもちまして、ニコニコアプリでのゲームサービスを終了させていただくこととなりました。

これまでエミル・クロニクル・オンラインをご愛顧くださいました皆様には、心より、厚く御礼申し上げます。

2017年5月25日（木）には、大型アップデートを実装いたします。  
引き続き、運営チーム一丸となって、全力で運営に取り組んでまいりますので、  
最後までエミル・クロニクル・オンラインをお楽しみくださいますよう、よろしくお願いいたします。

##### 今後の主なスケジュール

|  |  |
| --- | --- |
| 2017年5月15日（月）11：00 | ・EXTRA Shop Point付与終了**(\*1)** |
| 2017年5月18日（木）定期メンテナンス | ・ECO SHOP感謝セール**(\*2)**  ・PayNetCafe販売終了 |
| 2017年5月25日（木）定期メンテナンス | ・大型アップデート  ・エクストラアイテム（30日）のShop Point返還**（\*3）** |
| 2017年5月31日（水）定期メンテナンス開始まで | ・Shop Point販売終了  ・ECO公認店舗サービス終了 |
| 2017年5月31日（水）14：00 | ・ECOネットストア販売終了 |
| 2017年7月27日（木）10：00 | ・ゲームサービス終了 |

**(\*1)**EXTRA Shop Pointは、2,000ニコニコポイント以上のShop Pointをチャージした際に、おまけでもらえるShop Pointです。  
例）21,000 Shop Point（2,000ニコニコポイント）の内1,000 Shop Point分  
付与終了後は2,000ニコニコポイント=20,000Shop Pointとなります。  
     
**(\*2)**これまでのご愛顧に感謝し、ECO SHOPで販売しているエクストラアイテムをお手頃価格で提供させていただきます。  
   
**(\*3)**「2017年5月18日（木）定期メンテナンス時点」で、ワールドごとに4個以上所持している  
対象のエクストラアイテム（30日）については、サービス終了までに消費しきれないことから、  
消費可能な3個分を除いて、余った分を2017年5月25日（木）にShop Pointとしてお戻しさせていただきます。  
   
**▼対象アイテム**  
・有効期間が30日間の未使用のエクストラアイテム  
**※**対象アイテムの詳細は[**コチラ**](../../../../econline.gungho.jp/game_help/item/extraitem/itemlist.html)  
   
 例）5月18日(木)定期メンテナンス時点で未使用の経験値ふえーる５０(３０日)をFreesiaワールド で5個所持している場合  
**⇒**7月27日(木)までに消費可能な3個を除き、残りの2個分（販売価格15,000Shop Point×2」を、  
   5月25日(木)にShop Pointとしてお戻しいたします。  
   
**※**2017年5月18日（木）定期メンテナンス以降に、購入した分はShop Point返還の対象外となります。

サービス終了まで、皆様に楽しんでいただけますように様々な企画を予定しております。
