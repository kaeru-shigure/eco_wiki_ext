内容
:   キャンペーン期間中に、初めてエミル・クロニクル・オンライン（以下、「ECO」）をプレイして、イベントナビゲーションのチュートリアルを全てクリアした方の中から抽選200名様にECOゲーム内アイテム「ダンプティースーツ」をプレゼントいたします。

キャンペーン期間
:   2016年11月24日（木）定期メンテナンス終了　～　2016年12月22日（木）定期メンテナンス開始まで

対象条件
:   キャンペーン期間中に、初めてECOをプレイして、イベントナビゲーションのチュートリアルを全てクリアしたキャラクターをお持ちの方

| チュートリアルイベント名 | 開始方法 | クリアの確認方法 |
    | --- | --- | --- |
    | はじめてのおつかい | ［メニュー］ －［クエストナビインフォ］ －［イベントナビ］ －イベント名をクリックする | ［メニュー］ －［クエストナビインフォ］ －［イベントナビ］ －状態が［完了］になっている |
    | すてきなおくりもの |
    | だいじなおしごと |
    | たのしいなかま |
    | わくわくとたびだち |
    | ねがいとおもい |
    | たのしいおかいもの |
    | くらやみのぼうけん |
    | おしゃれとじつよう |
    | つたえたいつよさ |
    | こころざしのありか |
    | たびだちのとき |

- エントリーは1つのniconicoアカウントにつき、1回限りです。
    - 本キャンペーンの「ECOを初めてプレイ」とは、「エミル・クロニクル・オンライン利用規約」に同意して、ECOをプレイすることです。
    - 対象条件を満たすと、自動エントリーです。
    - niconicoアプリ初回利用時には、「ニコニコアプリ利用規約」に同意した上で、「エミル・クロニクル・オンライン利用規約」に同意して、ECOをプレイする必要があります。

景品
:   対象条件を満たした方の中から、抽選で以下の景品をプレゼントいたします。

| 景品 | 当選人数 |
    | --- | --- |
    | ゲーム内アイテム「ダンプティースーツ」1個 | 抽選で200名様 |

- 景品の当選者は、本キャンペーンにエントリーされた全ての方の中から抽選いたします。
    - 当選した方には、2017年1月下旬頃、［お知らせ］で景品のアイテムと交換可能なアイテムコードをご案内いたします。
    - ［お知らせ］は、ニコニコアプリ公式サイトの［お問い合わせ］-［ゲームに関するお問い合わせ］-［サポート履歴］から確認することができます。

注意事項
:   - 景品の配布までに、niconicoアカウントの解約などにより、ECOをご利用いただけなくなった場合、エントリーは無効となります。
    - お一人様によるエントリーならびに当選は1回限りです。同一のお客様であると判断した場合、2回目以降のエントリーならびに当選は無効となります。
    - エントリー状況や抽選結果等について、個別にお問い合わせいただいてもお答えできません。
    - 当ページの掲載事項およびエントリーに関する全ての条件は、予告なく変更する場合があります。あらかじめご了承ください。
