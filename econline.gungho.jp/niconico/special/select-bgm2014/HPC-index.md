# ECOヘヴィメタルアレンジBGM投票結果発表＆CD化決定！

皆様よりご投票いただいた、エミル・クロニクル・オンラインの世界を彩る数々のBGMの中からヘヴィメタルにアレンジして聞いてみたい楽曲の結果を発表いたします！
上位2曲をガンホーのヘヴィメタル音楽集団「G-SHINE」とニコニコ生放送で絶賛放送中の「月刊ファミ通feat.」の市野ルギア氏の率いる音楽ユニット「赤魂」がヘヴィメタルにアレンジ！
また、2014年冬に実装予定の「ドミニオン世界　都市攻略戦」に赤魂による完全オリジナルの新曲が登場することが決定いたしました！
さらに、上位2曲のアレンジBGMと都市攻防戦の楽曲をCDとしてエンターブレインのオンラインショッピングサイト「ebten」にて2014年10月31日（金）に発売!
初回限定生産には、特典としてパートナーの「ヘビメタイニー・アルマ」付き！

![ダウンタウンのシアター２で「エコメタ！」PV上映中！](../../../special/select-bgm2014/_img/downtown.png)

G-SHINE
:   ![G-SHINE](../../../special/select-bgm2014/_img/prof01.png)
:   2010年の「ラグナロクオンライン」エイプリルフール企画をきっかけに、音楽好きなガンホー社員で構成されたヘビィメタル音楽集団。  
    ラグナロクオンラインのパッケージ特典CD企画や、ECOのエイプリルフール、ガンホーフェスティバル2014のスペシャルLIVEなどタイトルを超えて活動。

赤魂（Red Soul）
:   ![赤玉](../../../special/select-bgm2014/_img/akadamalogo.png)
:   ファミ通feat.でもお馴染みの、ギタリストでゲームミュージックメタルアレンジャーな、市野ルギアが所属する音楽ユニット。  
    数々のオンラインゲームのメタルアレンジCDを手掛けている。
    「ゲーム音楽総メタル化計画」を発動。ECOの2013年エイプリルフール企画「ヘビメタイニーfeat.eb!Gshine」にも参加。
:   ゲームメタル音楽ユニット[「赤魂」公式サイト](http://akadama.redsoul.jp/)※外部リンクになります
