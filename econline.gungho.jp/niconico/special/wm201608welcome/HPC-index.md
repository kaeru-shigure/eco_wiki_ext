### キャンペーン内容

キャンペーン期間中に、初めてエミル・クロニクル・オンライン（以下、「ECO」）をプレイして、「イリスと記憶の書架」イベントを全てクリアした方の中から、抽選で53名様に素敵なプレゼントが当たります。

### キャンペーン期間

2016年7月21日（木）定期メンテナンス後　～　2016年8月18日（木）定期メンテナンス開始まで

### 対象条件

キャンペーン期間中に、初めてECOをプレイして、「イリスと記憶の書架」イベントを全てクリアしたキャラクターをお持ちの方

| イリスと記憶の書架イベント | 開始方法 | クリアの確認方法 |
| --- | --- | --- |
| 一の物語：盟約の吸血鬼 | アップタウン「黒の聖堂」側にある噴水近くにいる「元気が余っている女の子」に話をかける | ゲーム内メニューバー→「クエストナビインフォ」→「イベントナビゲーション」→「イリスと記憶の書架」 |
| 二の物語：燐火の蛇姫 |
| 三の物語：輪廻の魔女 |
| 四の物語：仔猫の神様 |
| 五の物語：海賊の宝物 |
| 六の物語：傾国の妖女 |
| 七の物語：恋慕の人魚姫 |
| 八の物語：賢者の黄金 |
| 九の物語：月夜の狼 |
| 十の物語：契約の悪魔 |
| 十一の物語：終幕の魔王 |
| 十二の物語：想送の流れ星 |
| 十三の物語：憶の書架 |

- ※対象条件を満たすと、自動エントリーです。
- ※エントリーは1つのniconicoアカウントにつき、1回限りです。
- ※本キャンペーンの「ECOを初めてプレイ」とは、「エミル・クロニクル・オンライン利用規約」に同意して、ECOをプレイすることです。
- ※niconicoアプリ初回利用時には、「ニコニコアプリ利用規約」に同意した上で、「エミル・クロニクル・オンライン利用規約」に同意して、ECOをプレイする必要があります。

### 景品

対象条件を満たした方の中から、抽選で53名様に下記の景品をプレゼントいたします。

| 景品 | 当選数 |
| --- | --- |
| 豪華サイン色紙（1枚） ※以下の人気声優さんのサイン寄せ書き 　・村川梨衣さん（清姫・ロア役） 　・佐藤聡美さん（玉藻・ロア役） | 3名様 |
| WebMoneyカード（500円分） | 50名様 |

- ※景品の当選者は、本キャンペーンにエントリーされた全ての方の中から抽選いたします。
- ※当選された方には、いずれかの景品をプレゼントいたします。景品の選択はできません。

### 景品の配布について

- 当選した方には、2016年9月上旬頃、［お知らせ］で景品受取方法に関するご案内をお送りいたします。［お知らせ］に記載の方法で、景品発送先をご連絡ください。
- ※［お知らせ］は、ニコニコアプリ公式サイトの［お問い合わせ］-［ゲームに関するお問い合わせ］-［サポート履歴］から確認することが出来ます。
- ※景品について、［お知らせ］に記載の期限までに、指定の方法で景品発送先をご連絡いただけない場合や、ご連絡いただいた景品発送先の不備、長期不在等により景品が主催者に返送された場合には、受け取りを辞退されたものと判断させていただきます。再発送はいたしません。
- ※発送は日本国内に限らせていただきます。
