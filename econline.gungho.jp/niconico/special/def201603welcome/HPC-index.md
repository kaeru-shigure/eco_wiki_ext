### キャンペーン内容

キャンペーン期間中に、初めてエミル・クロニクル・オンライン（以下、「ECO」）をプレイして、「イリスと記憶の書架」イベントを全てクリアした方の中から、抽選で3名様に、「ロア」のキャラクターボイスを担当いただいた人気声優さんたちのサインが入った「豪華サイン色紙」をプレゼントいたします。

### キャンペーン期間

2016年3月22日（火）20:00　～　2016年4月20日（水）定期メンテナンス開始まで

### 対象条件

キャンペーン期間中に、初めてECOをプレイして、「イリスと記憶の書架」イベントを全てクリアしたキャラクターをお持ちの方

| イリスと記憶の書架イベント | 開始方法 | クリアの確認方法 |
| --- | --- | --- |
| 第一巻：盟約の吸血鬼 | アップタウン「黒の聖堂」側にある噴水近くにいる「元気が余っている女の子」に話をかける | ゲーム内メニューバー→「クエストナビインフォ」→「イベントナビゲーション」→「イリスと記憶の書架」 |
| 第二巻：燐火の蛇姫 |
| 第三巻：輪廻の魔女 |
| 第四巻：仔猫の神様 |
| 第五巻：海賊の宝物 |
| 第六巻：傾国の妖女 |
| 第七巻：恋慕の人魚姫 |
| 第八巻：賢者の黄金 |
| 第九巻：月夜の狼 |
| 第十巻：契約の悪魔 |
| 第十一巻：終幕の魔王 |
| 第十二巻：想送の流れ星 |
| 最終巻：記憶の書架 |

- ※対象条件を満たすと、自動エントリーです。
- ※エントリーは1つのniconicoアカウントにつき、1回限りです。
- ※本キャンペーンの「ECOを初めてプレイ」とは、「エミル・クロニクル・オンライン利用規約」に同意して、ECOをプレイすることです。
- ※niconicoアプリ初回利用時には、「ニコニコアプリ利用規約」に同意した上で、「エミル・クロニクル・オンライン利用規約」に同意して、ECOをプレイする必要があります。

### 景品

対象条件を満たした方の中から、抽選で3名様に下記の景品をプレゼントいたします。

| 景品 | 当選者 |
| --- | --- |
| 豪華サイン色紙（1枚）    ※以下の人気声優さんのサインが全て入っています  ・南條愛乃さん（イリス・ロア役）  ・金元寿子さん（アルカード・ロア役）  ・門脇舞以さん（バステト・ロア役）  ・大空直美さん（ローレライ・ロア役）  ・悠木碧さん（メフィスト・ロア役）  ・小清水亜美さん（ルチフェロ・ロア役） | 3名様 |

- ※景品の当選者は、本キャンペーンにエントリーされた全ての方の中から抽選いたします。

### 景品の配布について

当選した方には、2016年5月上旬頃、［お知らせ］で景品受取方法に関するご案内をお送りいたします。［お知らせ］に記載の方法で、景品発送先をご連絡ください。

- ※［お知らせ］は、ニコニコアプリ公式サイトの［お問い合わせ］-［ゲームに関するお問い合わせ］-［サポート履歴］から確認することが出来ます。
- ※［お知らせ］に記載の期限までに、指定の方法で景品発送先をご連絡いただけない場合、景品の受け取りを辞退したものと判断させていただきます。
- ※発送は日本国内に限らせていただきます。

### 注意事項

- ※景品の配布までに、niconicoアカウントの解約などにより、ECOをご利用いただけなくなった場合、エントリーは無効となります。
- ※お一人様によるエントリーならびに当選は1回限りです。登録情報や景品送付先の情報などから同一のお客様であると判断した場合、2回目以降のエントリーならびに当選は無効となります。
- ※エントリー状況や抽選結果などについて、個別にお問い合わせいただいてもお答えできません。
