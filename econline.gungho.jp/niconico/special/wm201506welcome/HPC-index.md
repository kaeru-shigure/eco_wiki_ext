### キャンペーン内容

キャンペーン期間中に、初めてエミル・クロニクル・オンライン（以下：「ECO」）をプレイして、エミル、タイタニア、ドミニオンのいずれかのキャラクターレベルを15以上にした方の中から、抽選で合計1,000名様に、ゲーム内パートナーアイテム『ホノりん』『アワりん』『モリりん』『ヒカりん』『ワルりん』のいずれか1つをプレゼントいたします。

### キャンペーン期間

2015年6月25日（木）定期メンテナンス終了　～　2015年7月16日（木）定期メンテナンス開始

### 対象条件

キャンペーン期間中に、ECOを初めてプレイして、エミル、タイタニア、ドミニオンのいずれかのキャラクターレベルを15以上にした方

- ※ エントリーは1つのniconicoアカウントにつき1回限りです。
- ※ 本キャンペーンの「初めてECOをプレイ」とは、「エミル・クロニクル・オンライン利用規約」に同意して、ECOをプレイすることです。
- ※ niconicoアプリ初回利用時には、「ニコニコアプリ利用規約」に同意した上で、「エミル・クロニクル・オンライン利用規約」に同意して、ECOをプレイする必要があります。

### 景品

| 景品（ゲーム内パートナーアイテム） | 当選者数 |
| --- | --- |
| 『ホノりん』（1個） | 抽選で各200名様（合計1,000名様） |
| 『アワりん』（1個） |
| 『モリりん』（1個） |
| 『ヒカりん』（1個） |
| 『ワルりん』（1個） |

- ※ 景品の選択はできません。
- ※ 景品の当選者は、本キャンペーンにエントリーされた全ての方の中から抽選いたします。

### 景品の配布について

当選した方には、2015年8月中旬頃、［お知らせ］で景品のアイテムと交換可能なアイテムコードをご連絡いたします。

※［お知らせ］は、ニコニコアプリECO公式サイトの［お問い合わせ］の［ゲームに関するお問い合わせ］にある、［サポート履歴］から確認することができます。
