# 黒羽の矢が立つ芸術祭（くろはのやがたつげいじゅつさい）

![2014年10月23日（木）メンテナンス終了～2014年11月20日（木）メンテナンス開始まで](../../../special/termkurohanoya2014/_img/event/Read_cp_txt.png "2014年10月23日（木）メンテナンス終了～2014年11月20日（木）メンテナンス開始まで")

こんにちは。  
私はダークフェザー。この子はウィリー・ドゥよ。

普段はアミス先生の学校でお勉強しているんだけど、今は「作家志望の男」という人のお手伝いをしているの。  
この人は芸術祭までにお話を作るお仕事をしているのだけど、間に合いそうにないんですって。

私、お話を聞くのは好きだけど、自分で作るのはよくわからなくて……。  
この人は「話のネタを集める」って言うけど、「話のネタ」ってどんな形をしているのかしら。  
あなた、わかるかしら？

## 話のネタを報告しよう！

アルマ達から話を聞いたら、ダウンタウンにいる「作家志望の男」に教えてあげよう！  
新しい話のネタを教える度に、最初の1回だけ好きな場所に転送させてくれるよ！

※転送できる場所は、「鍵の契約」を行った場所です。  
※転送を断った場合、次の新しい話のネタを報告するまで転送サービスは受けられません。

締め切りは一週間ごとでアール！毎週新鮮なネタをプリーズでアール！

1回目
:   2014年10月23日（木）定期メンテナンス後～2014年10月29日（水）23：59まで

2回目
:   2014年10月30日（木）0：00～2014年11月5日（水）23：59まで

3回目
:   2014年11月6日（木）0：00～2014年11月12日（水）23：59まで

4回目
:   2014年11月13日（木）0：00～2014年11月20日（木）定期メンテナンス開始まで

## 話のネタを教えると…

毎週、締め切りまでに報告した話のネタの内容と数に応じた報酬がもらえます。

ネタの内容が珍しかったり、ネタの数が多いほどお礼は豪華になるのでアール！
