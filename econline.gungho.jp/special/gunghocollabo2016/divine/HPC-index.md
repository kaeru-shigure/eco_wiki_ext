# ECO×ディバインゲート

![コラボレーションイベントをプレイして、パートナー「聖王アーサー」を手に入れよう！](../../../special/gunghocollabo2016/divine/_img/pc/heroCatch.png "コラボレーションイベントをプレイして、パートナー「聖王アーサー」を手に入れよう！")

![ECOとは？](../../../special/gunghocollabo2016/divine/_img/pc/ecoTit.png "ECOとは？")
:   『エミル・クロニクル・オンライン』は、「ハートフル」がキーワードのオンラインRPG。  
    オンラインゲームがはじめての人でも簡単＆安心して遊べます。モンスターが想いの力で人の姿になった「アルマ」など個性的なキャラクターがたくさん登場！  
    オシャレを楽しんだり、自分だけの家を持てたり、季節イベントも盛り沢山。誰かのために頑張れる。そんなやさしい世界での冒険の旅が待ち受けるオンラインゲームです。

![コラボイベントをプレイするには？](../../../special/gunghocollabo2016/divine/_img/pc/eventTit.png "コラボイベントをプレイするには？")
:   「アップタウン」にいるNPC「アルカード・ロアに似た女の子」に話しかけて、出題される「ディバインゲートに関するクイズ」に答えよう！[詳しい操作法方法はコチラ](../../../manual/HPC-index.html)
:   クイズに正解すると、パートナー「聖王アーサー」をゲット！  
    クイズに不正解しても、ゲーム内アイテム「りんご」を1個を持って再度話しかけると挑戦できるよ！

![](../../../special/gunghocollabo2016/divine/_img/pc/eventPic.png)

![さらに！](../../../special/gunghocollabo2016/divine/_img/pc/spItemTit.png "さらに！")
:   パートナー「聖王アーサー」をLv30まで育てて装備した状態で「アルカード・ロアに似た女の子」に話しかけると特別なアイテムがもらえるよ！※受け取るには、アイテムインベントリに空枠3つ以上必要です。

![アクトキューブ「ホーリーグレイル（専）」](../../../special/gunghocollabo2016/divine/_img/pc/spItem1_f1.png "アクトキューブ「ホーリーグレイル（専）」")

アクトキューブ
:   「聖王アーサー」専用のアクトキューブ。攻撃スキル『ホーリーグレイル』を覚える。相手を凄い力で攻撃する。
:   [アクトキューブについてはコチラ](../../../gamedata/system/partner.html)

![アクトキューブ「ヒューマライズ：アサルトⅢ（専）」](../../../special/gunghocollabo2016/divine/_img/pc/spItem2_f1.png "アクトキューブ「ヒューマライズ：アサルトⅢ（専）」")

アクトキューブ
:   「聖王アーサー」専用のアクトキューブ。補助スキル『ヒューマライズ：アサルトⅢ』を覚える。自身の与えたダメージに一定時間ボーナスが加わる。
:   [アクトキューブについてはコチラ](../../../gamedata/system/partner.html)
