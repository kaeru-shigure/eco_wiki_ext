# 寒さを吹き飛ばせ☆ECOハートフルキャンペーン

【キャンペーン期間内容】

2013年1月24日（木）定期メンテナンス終了～2月14日（木）定期メンテナンス開始

キャンペーン期間中にECOのゲームアカウントを作成しかつ、ECOのゲーム内でエミル、タイタニア、ドミニオンいずれかのキャラクターをLV30以上にした方の中から、抽選で300名様にECOゲーム内アイテム「蝶の羽（ピンク）」をプレゼントいたします。

■対象条件
以下、（1）と（2）どちらかの条件を満たした方  
（1）キャンペーン期間中に、ECOのゲームアカウントを作成しかつ、エミル、タイタニア、ドミニオンいずれかのキャラクターをLV30以上にした方  
（2）ECO Light を一度も起動したことがないFacebookアカウントで、キャンペーン期間中にECO Lightを起動しかつ、エミル、タイタニア、ドミニオンいずれかのキャラクターをLV30以上にした方  
※エントリーは、1つのログインIDまたはFacebookアカウントにつき、1回限りです。  
≫ログインIDとは、ECO公式サイトやガンホーゲームズサイトにログインするためのIDです。ログインIDには、ガンホーIDの他、FacebookアカウントやGoogleアカウントなどが使用できます。ログインIDについては、[こちら](http://forum.gungho.jp/EokpControl?&tid=14791&event=QE0004)

■景品
ゲーム内アイテム『蝶の羽（ピンク）』（1個）　：　抽選で300名様  
※景品の当選者は、応募窓口に関わらず、本キャンペーンにエントリーされた全ての方の中から抽選いたします。

■景品の配布について
当選した方には、2013年3月中旬ごろ、ガンホーゲームズのサポートセンター「投稿確認」から確認可能な「お知らせ」で、景品のアイテムと交換可能なアイテムコードをご連絡いたします。  
※「お知らせ」の確認方法については、[こちら](http://forum.gungho.jp/EokpControl?&tid=17749&event=QE0004)をご確認ください。  
≫ECO Lightを遊んでいる方（OpenID未連携）  
ECO Lightをプレイいる方は、ガンホーゲームズで必要な情報を登録していただくことにより、「お知らせ」などのガンホーゲームズのサービスを利用することができます。登録方法については、[こちら](../../../econline.gungho.jp/support/oshirase/HPC-index.html)をご確認ください。

■注意事項
※対象条件を満たすと、自動的にエントリーされます。エントリーの取消はできません。  
※景品の配布までにログインIDの解約や対象のゲームアカウントの削除などにより、ECOをご利用できなくなった場合には、エントリーは無効となります。  
※エントリー状況や抽選結果などについて、個別にお問合せいただいてもお答えできません。
