# みんなで作ろう第16弾「ローディング画面イラストコンテスト」

## みなさんと共にECOを作っていく『みんなで作ろう！』の第16弾を開催いたします！ みんなで作ろう！第16弾では、プロ、アマ、ユーザーである、ないに関わらず“ECO”に関するイラストを自由にお描きいただき、 優秀作品は6月4日（土）にディファ有明にて開催される「ガンホーフェスティバル2011」の会場で展示いたします。 「ガンホーフェスティバル2011」にご来場いただいた方による投票によって【入賞】4作品を選出し、 ECOのゲーム内にローディング画面として実装いたします！ また、入賞作品受賞者には、『みんなで作ろう！』でのみ手に入れることができる 貴重なゲーム内アイテム「クリエイターマント」をプレゼント！ あなたの会心の一枚を是非、ご応募ください！

入賞作品受賞者には貴重な「**クリエイターマント**」をプレゼント！

「ガンホーフェスティバル2011」会場で投票していただいた方には、
ゲーム内アイテム「**特製プロモーションイリスカード**」プレゼント！

### 応募要項

#### コンテスト概要

ECOゲームクライアントのローディング画面のイラストを募集します。
応募期間中に、エントリーセンター内専用フォームよりご応募ください。
応募作品の中からECO運営チームが選定した作品を2011年4月29日（金・祝）に
東京ビックサイトで行われる「RJC2011＆ガンホーフェスティバル2011」の会場で展示し、
来場された方に気に入った作品を投票していただきます。

投票締め切り後に投票結果を集計し、上位入賞作品を発表いたします。

入賞作品は、ECOゲームクライアントのローディング画面に採用させていただき、
入賞作品を応募された方にはECOゲーム内アイテム「クリエイターマント」をプレゼントいたします。

また、「RJC2011＆ガンホーフェスティバル2011」の会場で投票をしていただいた方には、
もれなくECOゲーム内アイテム「プロモーションイリスカード」をプレゼントいたします。

### みんなで作ろう第16弾「ローディング画面イラストコンテスト」Q&A

Q01イラストは、カラーで描いた方がいいのでしょうか？モノクロで描きたいのですが。
:   イラストは、カラーでもモノクロでも構いません。作品のイメージにあった色でお描きいただければ結構です。

Q02描いてはいけないものはありますか？
:   基本的にゲーム内に実装されているアイテムや衣装などであれば、そのまま描いていただくことは問題ありません。  
    しかしゲーム内に実装されていない国旗やシンボルマーク、ロゴマークなどをイラストに取り入れることはおやめください。

Q03手書きの作品をスキャンして応募してもいいですか？
:   手書き作品をスキャンした場合でも、作品規定に沿ったデータであれば応募可能です。  
    ただし、データ化した状態でイラストが判別できない場合、選考対象外となりますので、鉛筆書きなどはおやめください。

Q04郵送やメールでも応募できますか？
:   エントリーセンターのみでの受付となります。

Q05作品規定で「横書き：縦600×横800ピクセル  解像度：72dpi」と決められているのは、どうしてですか？
:   本コンテストでは、応募作品のうち、ECO運営チームが選定した作品を印刷し、
    「RJC2011＆ガンホーフェスティバル2011」の会場で展示します。  
    また、「入賞」作品はECOゲームクライアントのローディング画面として使用させていただきます。  
    そのため、印刷やゲーム内での使用に適しているサイズを決めさせていただいております。

Q06応募した作品を描き直したいのですが。
:   一度ご応募いただいた作品や応募内容は、取り消し、変更することはできません。

Q07作品の中にNPCやモンスターは必ず描かなければいけませんか？
:   ECOに関するイラストであれば、NPCやモンスターが描かれていなくても問題ありません。

Q08イラスト内にセリフやコメントを入れてもいいですか？
:   「入賞」の作品はECOゲームクライアントのローディング画面として使用させていただくため、
    セリフやコメントなど文字の挿入はお止めください。
