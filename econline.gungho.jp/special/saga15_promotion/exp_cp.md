## 経験値1.5倍キャンペーン開催!!

大型アップデート「SAGA15：永遠の絆」の実装で、さらなる広がりをみせるECOワールド。

「もっと楽しみたいけど、レベルがなぁ・・・」なんて悩んでいる冒険者の方に朗報！経験値が通常よりお得にゲットできる「**経験値1.5倍キャンペーン**」を開催いたします！

キャンペーン期間中、モンスターを倒した時の経験値と職業経験値が、なんと通常の**1.5**倍に！　獲得経験値が通常より高い**ECO公認ネットカフェ**でプレイすれば、もっとお得に経験値がゲットできちゃう!!

ECOをもっともっと楽しみたいベテラン冒険者もECO新米冒険者も、この機会にレベルを上げて強くなろう!!

期間中、モンスターを倒した際に獲得できる経験値、職業経験値が増加します。

### 最終獲得経験値を計算早見表でチェックしよう!!（基本経験値を100%とした場合）

| 経験値リスト | モンスターを倒した時 | | クエスト | |
| --- | --- | --- | --- | --- |
| 経験値 | 職業経験値 | 経験値 | 職業経験値 |
| (1)アイテム・特典を使用していない場合の基本獲得経験値（標準）→**経験値1.5倍キャンペーン中** | 100%→**150%** | 100%→**150%** | 100% | 100% |
| (2) 「経験値ふえーる」系アイテムの効果による増加経験値 | 50%/100%/150%/300% | 50%/100%/150%/300% | - | - |
| (3) 「職業経験値ふえーる」系アイテムの効果による増加経験値 | - | 50%/100%/150%/300% | - | - |
| (4) 公認ネットカフェ特典（獲得経験値50％UP）の効果による増加経験値 | 50% | 50% | - | - |
| (5) スキル「エキスパンド」を使用した場合の増加経験値 | 5% | 5% | 5% | 5% |
| アイテム・特典を全て使用した場合の最終獲得経験値 {(1)+(2)+(3)+(4)}×(5) | 262.5%～525% | 315%～840% | 105% | 105% |

計算したい基本獲得経験値の列にある(1)に、該当する(2)～(5)の経験値を足すことにより、基本獲得経験値に対して、アイテム・特典を使用した際に得られる最終獲得経験値が計算されます。

なお、プライベートダンジョンは"標準"でディメンションダンジョンより多めの経験値が設定されています。プライベートダンジョンで特典を使用した場合は、上記の表が適用されます。
