## キャンペーン概要

![](../../special/saga17/_img/common/waku_top-left.png)![](../../special/saga17/_img/common/waku_top-right.png)

【キャンペーン期間】  
2012年9月6日（木）定期メンテナンス終了～2012年9月20日（木）定期メンテナンス開始

【キャンペーン概要】  
期間中、モンスターを倒した際に獲得できる基本経験値、職業経験値が増加します。  
※クエストによる基本経験値・職業経験値は増加しませんのでご注意ください。

## ECO公認ネットカフェなら経験値はもっとお得に！

![最終獲得経験値を計算早見表でチェックしよう！](../../special/saga17/_img/experience/h2-list.png "最終獲得経験値を計算早見表でチェックしよう！")

| 経験値リスト | モンスター | | クエスト | |
| --- | --- | --- | --- | --- |
| 経験値 | 職業経験値 | 経験値 | 職業経験値 |
| (1)アイテム・特典を使用していない場合の基本獲得経験値(標準) →経験値1.5倍キャンペーン中 | 100%→150% | 100%→150% | 100% | 100% |
| (2)「経験値ふえーる」系アイテムの効果による増加経験値 | 50%/100%/150%/ 300% | 50%/100%/150%/ 300% | - | - |
| (3)「職業経験値ふえーる」系アイテムの効果による増加経験値 | - | - | - | - |
| (4)公認ネットカフェ特典 （獲得経験値50％UP）の効果による増加経験値 | 50% | 50% | - | - |
| (5)期間中にリサイクルシステムの経験値アップ効果が発生した場合の増加経験値 | 50% | 50% | - | - |
| (6)スキル「エキスパンド」を使用した場合の増加経験値 | 5% | 5% | 5% | 5% |
| アイテム・特典を全て使用した場合の最終獲得経験値 {(1)+(2)+(3)+(4)+(5)}×(6) | 315%～577.5％ | 315%～577.5％ | 105% | 105% |

![](../../special/saga17/_img/common/waku_top-left.png)![](../../special/saga17/_img/common/waku_top-right.png)

計算したい基本獲得経験値の列にある(1)に、該当する(2)～(5)の経験値倍率を加算、(6)は乗算することにより、基本獲得経験値に対して、アイテム・特典を使用した際に得られる最終獲得経験値が計算されます。  
なお、プライベートダンジョンは"標準"でディメンションダンジョンより多めの経験値が設定されています。  
プライベートダンジョンで特典を使用した場合は、上記の表が適用されます。
