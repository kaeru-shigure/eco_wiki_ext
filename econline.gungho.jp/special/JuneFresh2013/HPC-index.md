# 水も滴るフレッシュ！ECOスタートキャンペーン

![ECOスタートキャンペーン](_img/lead.png)

[![今すぐECOを始めるにはこちら](_img/btn_ecostart_f1.png)](../../../econline.gungho.jp/ecolightgg/lp/HPC-index.html)

※ECOスタートキャンペーンにご参加いただくためには、ゲーム終了時に選択できる「明日も遊ぶ」をクリックし  
無料会員登録をしていただく必要がございます。
詳細は下記をご覧ください。

![「LV30以下無料支援アイテム」が好きなだけ使い放題！](_img/start_img01.png)

●キャンペーン内容
:   キャンペーン期間中に、初めてECO（ECO Light GG、ECO Lightを含む）をプレイして、エミル、タイタニア、ドミニオンいずれかのキャラクターをLV30以上にした方の中から、  
    抽選で500名様にゲーム内アイテム『雨雲スイッチ』をプレゼントいたします。

●キャンペーン期間
:   2013年6月27日（木）定期メンテナンス終了～2013年7月18日（木）定期メンテナンス開始

●対象条件
:   以下、【A】のいずれかの条件と【B】の条件を満たした方
    **【A】**
    以下、（1）～（3）いずれかの条件を満たした方

- （1）キャンペーン期間中に、ECOのゲームアカウントを作成したログインIDをお持ちの方
    - （2）キャンペーン期間中に、ECO Light GGのお試しプレイをした後、ECOのゲームアカウント作成とログインID登録をした方
    - （3）キャンペーン期間中に、OpenID未連携のFacebookアカウントで、初めてECO Light を起動した方

**【B】**
    キャンペーン期間中に【A】の条件を満たしたゲームアカウントまたはFacebookアカウントで、エミル、タイタニア、ドミニオンいずれかのキャラクターをLV30以上にした方

- ※【A】（2）の「ECO Light GGのお試しプレイをした後、ECOのゲームアカウント作成とログインID登録」とは、「明日も遊ぶ」をクリックして、無料会員登録をすることです。
    - ※ECO Light GGのお試しプレイ後、ログインIDの登録をしない場合、作成したキャラクターやゲームをプレイした情報は削除されます。  
      また、プレイ中に、メンテナンス等によりゲームから切断された場合やブラウザを閉じた場合も、情報は削除されます。
    - ※ECO Light GGのお試しプレイで、エミル、タイタニア、ドミニオンいずれかのキャラクターをLV30以上にした後に、  
      ECOのゲームアカウント作成とログインID登録をした場合も、本キャンペーンの対象になります。
    - ※複数のゲームアカウント、キャラクターで対象条件を満たしても、エントリーは1つのログインIDまたはFacebookアカウントにつき、1回限りです。
    - ≫ログインIDとは、ECO公式サイトやガンホーゲームズサイトにログインするためのIDです。  
      ログインIDには、ガンホーIDの他、FacebookアカウントやGoogleアカウントなどが使用できます。ログインIDについては、[こちら](http://forum.gungho.jp/EokpControl?&tid=14791&event=QE0004)

●景品
:   ECOゲーム内アイテム『雨雲スイッチ』（1個）　：　抽選で500名様

- ※景品の当選者は、応募窓口に関わらず、本キャンペーンにエントリーされた全ての方の中から抽選いたします。

●景品の配布について
:   当選した方には、2013年8月下旬ごろ、ガンホーゲームズのサポートセンター「投稿確認」から確認可能な「お知らせ」で、  
    景品のアイテムと交換可能なアイテムコードをご連絡いたします。

- ※「お知らせ」の確認方法については、[こちら](http://forum.gungho.jp/EokpControl?&tid=17749&event=QE0004)をご確認ください。
    - ≫ECO Lightを遊んでいる方（OpenID未連携）
      ECO Lightをプレイいる方は、ガンホーゲームズで必要な情報を登録していただくことにより、  
      「お知らせ」などのガンホーゲームズのサービスを利用することができます。  
      登録方法については、[こちら](../../support/oshirase/HPC-index.html)をご確認ください。

●注意事項
:   - ※対象条件を満たすと、自動的にエントリーされます。エントリーの取消はできません。
    - ※景品の配布までにログインIDの解約や対象のゲームアカウントの削除などにより、ECOをご利用できなくなった場合には、エントリーは無効となります。
    - ※エントリー状況や抽選結果などについて、個別にお問合せいただいてもお答えできません。
    - ※当ページ掲載事項およびエントリーに関する全ての条件は、予告なく変更する場合があります。あらかじめご了承ください。
