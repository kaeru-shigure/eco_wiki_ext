## ドミニオン世界ですでに育てているキャラクターについて

ドミニオン世界の変更に伴い、すでにドミニオン世界で育てているキャラクターについては下記のご対応をさせていただきます。

#### 対象となるキャラクター

2011年8月25日（木）定期メンテナンスまでに、ドミニオン世界に「次元転生」したことのあるキャラクター

#### 実施日

2011年9月1日（木）定期メンテナンス時に、ゲーム内「ギフトシステム」にてアイテムを各キャラクターに配布いたします。  
ギフトシステムについて詳しくは[こちら](../../manual/ecoshop.html)をご覧ください。

【注意】  
ギフトシステムについては、一度に受信できる数が10通までとなっておりますので、 アイテムをご確認いただけないユーザー様は、お手数ではございますが以下手順でご確認くださいますようお願いいたします。

**≪アイテム確認手順≫**

1. ギフトウィンドウで現在受信しているアイテムを受け取り、メッセージを削除
2. 一旦ゲームからログアウト
3. 再度ゲームへログインしギフトウィンドウを開く

#### 「ドミニオン十字勲章」「ドミニオン黒竜十字勲章」「ドミニオン名誉勲章」「ドミニオン黒竜名誉勲章」について

これらのアイテムは、「使用した時点でのキャラクターレベル（名誉勲章の場合はジョブレベル）を元に、  
その次のレベルになるまでに必要な経験値のX％を得ることができる」という効果を持ちます。  
※使用時のレベルによって、何％の経験値を得られるかは以下をご確認ください。

- ドミニオン十字勲章

| 使用時の Baseレベル | アイテムの効果 |
  | --- | --- |
  | LV80以下 | 経験値50％増加 |
  | LV81～LV90 | 経験値30％増加 |
  | LV91～LV100 | 経験値20％増加 |
  | LV101～LV110 | 経験値10％増加 |
- ドミニオン黒竜十字勲章

| 使用時のBase  レベル | アイテムの効果 |
  | --- | --- |
  | LV80以下 | 経験値100％増加 |
  | LV81～LV90 | 経験値75％増加 |
  | LV91～LV100 | 経験値50％増加 |
  | LV101～LV110 | 経験値25％増加 |
- ドミニオン名誉勲章

| 使用時の  Jobレベル | アイテムの効果 |
  | --- | --- |
  | LV20以下 | 職業経験値50％増加 |
  | LV21～LV30 | 職業経験値30％増加 |
  | LV31～LV40 | 職業経験値20％増加 |
  | LV41以上 | 職業経験値10％増加 |
- ドミニオン黒竜名誉勲章

| 使用時のJobレベル | アイテムの効果 |
  | --- | --- |
  | LV20以下 | 職業経験値100％増加 |
  | LV21～LV30 | 職業経験値75％増加 |
  | LV31～LV40 | 職業経験値50％増加 |
  | LV41以上 | 職業経験値25％増加 |

> 例として、Jobレベル30のエミルの冒険者が「ドミニオン名誉勲章」を使用します。  
> この時、このキャラクターには「Jobレベル31になるために必要な経験値の30％」が追加されます。
>
> このアイテムの利用タイミングによって、以下の３つのパターンがあります。

**1 : 利用した時に、レベルが上がらない**

この冒険者が「Jobレベル30で、現在20％職業経験値がたまっている」場合、  
「Jobレベル31になるために必要な職業経験値合計」の30％分の職業経験値が追加されて、結果50％となります。

![1 : 利用した時に、レベルが上がらない](img/about_character/illust01.gif)
**2 : 利用した時に、次のレベルに上がる**

この冒険者が「Jobレベル30で、現在80％職業経験値がたまっている」場合、  
まず「Jobレベル30になるために必要な職業経験値合計」の30％分の職業経験値が追加されて、  
そのうち20％分でJobレベル31に到達します。残りの10％分は切り捨てにはならず、そのままの数値が加算されますが、「Jobレベル32になるために必要な職業経験値の10％ではない」ため、％表示は必ずしも10％にはなりません。

![2 : 利用した時に、次のレベルに上がる](img/about_character/illust02.gif)
**3 : 利用した時に、レベルキャップに到達してしまう**

もしこの冒険者が「ガーデナー／ブリーダーなどの、Jobレベル30がレベルキャップとなっている職業についていて、なおかつ現在80％経験値がたまっている」場合。この場合は、溢れた10％分の職業経験値は切り捨てとなります。

![3 : 利用した時に、レベルキャップに到達してしまう](img/about_character/illust03.gif)

#### 「ドミニオン機兵勲章」「ドミニオン機竜勲章」について

これらのアイテムは、「使用した時点でのコストリミット値を元に、一定量のコストリミットを増加させる」という効果を持ちます。

- ドミニオン機兵勲章

| 使用時の  Baseレベル | アイテムの効果 |
  | --- | --- |
  | LV80以下 | CL4増加 |
  | LV81～LV90 | CL3増加 |
  | LV91～LV100 | CL2増加 |
  | LV101～LV110 | CL1増加 |
- ドミニオン機竜勲章

| 使用時のBaseレベル | アイテムの効果 |
  | --- | --- |
  | LV80以下 | CL5増加 |
  | LV81～LV90 | CL4増加 |
  | LV91～LV100 | CL3増加 |
  | LV101～LV110 | CL2増加 |

コストリミットを増加させた際に、コストリミットの最大値を超えてしまう場合は
その分のコストリミット増加量が切り捨てとなってしまいます。

#### 「ドミニオン世界の記憶箱」について

![SS](img/about_character/image_ss.jpg)

「ドミニオン世界の記憶箱」は開封するとそれぞれ箱に対応したアイテムを入手することができます。  
入手できるアイテムの内容は「ドミニオン世界の記憶箱」のランクによって異なります。

**開封時に入手できるアイテム**

- 「ドミニオン世界の記憶箱（銅）」

  【ドミニオンセット】

  エンシェントギア、オブリージュ（上）、オブリージュ（下）

  ドミニオンセット、DEMセットのアイテム全てを入手できます。
