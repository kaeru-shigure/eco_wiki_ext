# Anemoneワールド変更・調整内容

## 攻撃速度（A.SPD）について

通常の攻撃速度が当初の想定よりも強力になっており、
スキル実装やモンスターの設定など遊び方に大きな影響を及ぼしていることを踏まえ、
新しい遊び方を提案できるよう、以下のような攻撃速度の調整・変更を行います。

- A.SPDの上限値を現状の「960」から「800」へと変更いたします。
- 詠唱時間がA.SPDに依存しているスキルの詠唱時間の下限を撤廃いたします。
- ステータス上昇によるS.AVOID・L.AVOIDおよびS.HIT・L.HITの上限値を現状の「400」から「500」へと引き上げをいたします。
- 攻撃の命中判定時に、LV差補正とキャラクターまたはペットが3体以上のモンスターにターゲットされている場合による命中率上昇が行われても  
  AVOIDの値によって軽減できるよう変更をいたします。
- 攻撃速度の調整に伴い、攻撃速度に影響のあるスキルの変更・調整をいたします。
- 爪の装備に装備時パッシブスキル「クリティカル率アップ（小）」を追加いたします。

※アイテム「冥王爪・オルトロス」はパッシブスキル「クリティカル率アップ」のみです。
- 爪、二丁拳銃を装備した際の A.SPD 減少効果を 7/10 で行うように変更いたします。
- 爪、二丁拳銃、一部スキルなどの攻撃回数の増加分を除き、通常攻撃を1秒間で7回までに制限いたします。
- 爪、二丁拳銃を装備した際に AGI によって A.SPD の上限値「800」まで達することができるよう変更いたします。

### 攻撃速度（A.SPD）の調整に伴い、変更・調整が生じるスキルについて

攻撃速度の調整に伴い、攻撃速度に影響を与える以下のスキルの変更・調整を行います。

| イレイザー | AVOIDコミュニオン | S.AVOID、L.AVOIDの上昇値を下方修正いたします。 |

| ソーサラー | オーバーワーク | 対象の魔法のディレイを短縮するスキルへと変更いたします。  魔法ディレイキャンセル効果以外のディレイキャンセル効果と重複できるように変更いたします。 |
| エンチャンター | パワーエンチャント | スピードエンチャントの仕様変更に伴い、物理攻撃速度・魔法詠唱速度・回避率・魔法抵抗率の低下を緩和いたします。 |
| スピードエンチャント | 対象の物理攻撃力・物理防御力・魔法攻撃力・魔法防御力が低下する効果と攻撃速度の上昇効果を撤廃し、対象の近距離回避・魔法抵抗率・魔法詠唱速度の上昇とディレイキャンセル効果を適用するスキルへと変更いたします。 |
