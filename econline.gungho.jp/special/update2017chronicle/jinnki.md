# 決戦～破滅の黄風

## あらゆる技術、力、エネルギーを集結させた希望を切り開く力「神器」

脅威的な力を持ったハスター、そして覚醒するクトゥルフに対抗するには、これまでにない力が必要不可欠。  
世界のすべての英知と技術、力を結集させ、究極の武器――神器を手に入れよう。

- :   ![耐久度なし！](../../special/update2017chronicle/_img/jinnki/eventPic01.png "耐久度なし！")

耐久度なし！
  :   究極の武器である「神器」には耐久度がなく、通常の戦闘で壊れることはありません。ただし、武具製錬、スロット拡張などで失敗すると消失いたします。
- :   ![スキル「竜眼開放・神」](../../special/update2017chronicle/_img/jinnki/eventPic02.png "スキル「竜眼開放・神」")

スキル「竜眼開放・神」
  :   神器を入手すると使えるようになる新スキル「竜眼開放・神」。
- :   ![初期開放スロット5個！](../../special/update2017chronicle/_img/jinnki/eventPic03.png "初期開放スロット5個！")

初期開放スロット5個！
  :   イリスカードスロットが初期状態で5個開放されているので、すぐにイリスカードをセットすることができます。
- 入手イベント開始条件
  :   - メインストーリー「破滅の黄風」をクリア

イベントの開始方法
  :   *三種族*条件を満たした状態で、イベントナビ「職服イベント２」⇒「LV120武器～神器解放～」をスタート
  :   *DEM種族*条件を満たした状態で、イベントナビ「DEMイベント」⇒「新セットパーツ入手～神器解放～」をスタート

※神器は「三種族用」と「DEM種族用」で分かれます。異なる種族の神器イベントを進めることはできません。
