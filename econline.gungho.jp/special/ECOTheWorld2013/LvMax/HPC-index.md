# いきなり最強レベル！

![レベル110の三次職や機械種族で遊ぶことができちゃう！いきなり最強レベル！](../../../special/ECOTheWorld2013/LvMax/_img/main_tit.png "レベル110の三次職や機械種族で遊ぶことができちゃう！いきなり最強レベル！")

![](../../../special/ECOTheWorld2013/LvMax/_img/LvMax_pic01.png)

「なんでもECO！ザ・ワールド」ではレベル上げ不要の最強キャラクターをすぐに楽しめちゃいます！

方法は簡単！キャラクターを作成し、ゲーム内NPCの指示どおりに進めると最強レベルのキャラクターになれちゃいます！

## まずはこの子に話しかけよう！

一番初めの世界で出会う女の子「ティタ」に話しかけてください。

会話を進めると、自分のキャラクターが最強状態になっています。

もちろんＤＥＭ（機械種族）のキャラクターもいきなりLV・コストリミットがMAXのキャラクターで始められます。

## 職業を選ぼう

エミル族・タイタニア族・ドミニオン族で「ティタ」と会話を進めると職業選択をすることになります。

気になる職業をえらんでみてください。後から職業変更をすることも可能です。

[![職業をチェック](../../../special/ECOTheWorld2013/LvMax/_img/btn01_f1.png "職業をチェック")](../../../gamedata/job/gladiator.html)
[![DEM装備をチェック](../../../special/ECOTheWorld2013/LvMax/_img/btn02_f1.png "DEM装備をチェック")](../../../gamedata/item/extraitem/demparts.html)

## 装備やアイテムをそろえよう！

南アクロニア平原のNPCから、いろんな装備やアイテムを手に入れてキャラクターをパワーアップしちゃおう！

ステータスを設定するのを忘れずにね！

![](../../../special/ECOTheWorld2013/LvMax/_img/pic05.png)
![貰えるアイテムがNPCによって違うからよく確認してね](../../../special/ECOTheWorld2013/LvMax/_img/pic06.png "貰えるアイテムがNPCによって違うからよく確認してね")
