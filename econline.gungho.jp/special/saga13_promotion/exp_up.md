## 経験値UPキャンペーン

これまでECOを遊んでいただいている方はもちろん、今回初めてECOを体験される方にも、「SAGA13：拓かれる大地」の実装によってますます遊びやすくなりました。  
「ECOの世界をもっと体験したいけど、なかなか強くならなくて…」とお悩みの貴方のために、「経験値UPキャンペーン」を開催いたします！   
2011年4月14日（木）～2011年4月28日（木）はモンスターを倒した時に獲得できる経験値と職業経験値が通常の1.5倍、2011年4月28日（木）～2011年5月12日（木）には、ECO公認ネットカフェからのみモンスター討伐経験値および職業経験値が通常特典の1.5倍から2倍に!!  
この機会にどんどんレベルアップして強くなっちゃおう！

![経験値UPキャンペーン](../../special/saga13_promotion/img/ss_02.jpg)

**イベント期間**  
**経験値1.5倍キャンペーン：**  
2011年4月14日（木）定期メンテナンス終了後～2011年4月28日（木）定期メンテナンス開始  
**ECO公認ネットカフェ経験値2倍キャンペーン：**  
2011年4月28日（木）定期メンテナンス終了後～2011年5月12日（木）定期メンテナンス開始

### ECO公認ネットカフェだと、期間中は獲得経験値が2倍に！さらにエクストラアイテムを使えば・・・？！

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 経験値リスト | モンスター | | クエスト | |
| 経験値 | 職業経験値 | 経験値 | 職業経験値 |
| (1)アイテム・特典を使用していない場合の基本獲得経験値（標準）→経験値1.5倍キャンペーン中 | 100%→150% | 100%→150% | 100% | 100% |
| (2) 「経験値ふえーる」系アイテムの効果による増加経験値 | 50%/100%/150%/300% | 50%/100%/150%/300% | - | - |
| (3) 「職業経験値ふえーる」系アイテムの効果による増加経験値 | - | 50%/100%/150%/300% | - | - |
| (4) 公認ネットカフェ特典（獲得経験値100％UP）の効果による増加経験値→ECO公認ネットカフェ経験値2倍キャンペーン中 | 50%→100% | 50%→100% | - | - |
| (5) スキル「エキスパンド」を使用した場合の増加経験値 | 5% | 5% | 5% | 5% |
| アイテム・特典を全て使用した場合の最終獲得経験値 {(1)+(2)+(3)+(4)}×(5) | 262.5%～525％ | 315%～840％ | 105% | 105% |

計算したい基本獲得経験値の列にある(1)に、該当する(2)～(5)の経験値を足すことにより、基本獲得経験値に対して、アイテム・特典を使用した際に得られる最終獲得経験値が計算されます。  
なお、プライベートダンジョンは"標準"でディメンションダンジョンより多めの経験値が設定されています。プライベートダンジョンで特典を使用した場合は、上記の表が適用されます。  
また、公認ネットカフェ経験値2倍キャンペーン中の基本獲得経験値は100%になります。

### ECO公認ネットカフェだと、期間中は獲得経験値が2倍に！さらにエクストラアイテムを使えば・・・？！

全国にあるECO公認ネットカフェだと、キャンペーン効果にネットカフェ特典である1.5倍が加わり、獲得できる経験値が通常の2倍に！  
そのうえ、エクストラアイテム「経験値ふえーる」「職業経験値ふえーる」などを併用すれば、さらに多くの経験値を得ることも可能です!!
