### キャンペーン内容

キャンペーン期間中に、初めてエミル・クロニクル・オンライン（ECO Light GGを含む。以下、「ECO」）をプレイして、「イリスと記憶の書架」イベントを全てクリアした方の中から、抽選で3名様に、「ロア」のキャラクターボイスを担当いただいた人気声優さんたちのサインが入った「豪華サイン色紙」をプレゼントいたします。さらに、抽選で100名様にECOゲーム内アイテム「デビューへの紹介状」をプレゼントいたします。

### キャンペーン期間

2016年5月26日（木）定期メンテナンス後　～　2016年6月16日（木）定期メンテナンスまで

### 対象条件

キャンペーン期間中に、初めてECOをプレイして、「イリスと記憶の書架」イベントを全てクリアしたキャラクターをお持ちの方

| イリスと記憶の書架イベント | 開始方法 | クリアの確認方法 |
| --- | --- | --- |
| 一の物語：盟約の吸血鬼 | アップタウン「黒の聖堂」側にある噴水近くにいる「元気が余っている女の子」に話をかける | ゲーム内メニューバー→「クエストナビインフォ」→「イベントナビゲーション」→「イリスと記憶の書架」 |
| 二の物語：燐火の蛇姫 |
| 三の物語：輪廻の魔女 |
| 四の物語：仔猫の神様 |
| 五の物語：海賊の宝物 |
| 六の物語：傾国の妖女 |
| 七の物語：恋慕の人魚姫 |
| 八の物語：賢者の黄金 |
| 九の物語：月夜の狼 |
| 十の物語：契約の悪魔 |
| 十一の物語：終幕の魔王 |
| 十二の物語：想送の流れ星 |
| 最終巻：記憶の書架 |

- ※本キャンペーンの「初めてECOをプレイ」とは、ECOゲームアカウントを作成し、ECOをプレイすることです。
- ※ECO Light GGのお試しプレイで、対象条件を満たした後に、ECOのゲームアカウント作成とログインIDを登録した場合も本キャンペーンの対象になります。
- ※ECO Light GGのお試しプレイ後、ログインIDを登録しない場合、作成したキャラクターやプレイした情報は削除されます。また、プレイ中にメンテナンス等によりゲームから切断された場合やブラウザを閉じた場合も、情報は削除されます。
- ※対象条件を満たすと、自動エントリーです。
- ※複数のゲームアカウント、キャラクターで対象条件を満たしても、エントリーは1つのログインIDにつき、1回限りです。

### 景品

対象条件を満たした方の中から、抽選で下記の景品をプレゼントいたします。

| コース | 景品 | 当選人数 |
| --- | --- | --- |
| A | 豪華サイン色紙（1枚） ※以下の声優さんのサインが全て入っています ・前野智昭さん（ワーウルフ・ロア役） ・折笠富美子さん（パラケルスス・ロア役） | 3名様 |
| B | ゲーム内アイテム「デビューへの紹介状」（1個） | 100名様 |

- ※当選はいずれかのコースのみです。コース、景品を選ぶことはできません。
- ※景品の当選者は、本キャンペーンにエントリーされた全ての方の中から抽選いたします。
