キャンペーン内容
:   2014年9月25日（木）定期メンテナンス終了～2015年3月26日（木）定期メンテナンス開始までの期間に、初めてエミル・クロニクル・オンライン（ECO Light GGを含む、以下「ECO」）にログインし、かつ2015年3月26日（木）定期メンテナンス終了～2015年4月9日（木）定期メンテナンス開始までの期間に、一度もECOにログインをしていない方に、もれなくECOゲーム内アイテム『信頼度ふえーる4H』と『EXパッシブアクトキューブBOX1』をプレゼントいたします。

対象
:   2014年9月25日（木）定期メンテナンス終了時点でECOのゲームアカウントをお持ちでないログインIDで、以下(1)(2)条件を満たすゲームアカウント
:   |  | 対象条件 |
    | --- | --- |
    | (1) | 2014年9月25日（木）定期メンテナンス終了～2015年3月26日（木）定期メンテナンス開始までの期間に、初めてECOにログインしたゲームアカウント |
    | (2) | 2015年3月26日（木）定期メンテナンス終了～2015年4月9日（木）定期メンテナンス開始までの期間に、一度もECOにログインをしていないゲームアカウント |

- 「初めてECOにログイン」とは、新たにゲームアカウントを作成し、ECOのゲーム内マップにキャラクターが表示された時点のことです。
    - 「ECOにログイン」とは、ECOのゲーム内マップにキャラクターが表示された時点のことです。
    - エントリーは、ゲームアカウント単位です。対象条件を満たすと、自動エントリーされます。

景品
:   | 景品（ゲーム内アイテム） | 配布対象 |
    | --- | --- |
    | 『信頼度ふえーる4H』（1個） 『EXパッシブアクトキューブBOX1』（1個） | 対象者全員 |

- 複数のゲームアカウントが対象となった場合も、景品の配布は1つのログインIDにつき1回限りです。
