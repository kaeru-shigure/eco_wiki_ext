内容
:   キャンペーン期間中に、初めてエミル・クロニクル・オンライン（ECO Light GGを含む。以下、「ECO」）をプレイして、イベントナビゲーションのチュートリアルを全てクリアした方の中から抽選200名様にECOゲーム内アイテム「ダンプティースーツ」をプレゼントいたします。

キャンペーン期間
:   2016年11月24日（木）定期メンテナンス終了　～　2016年12月22日（木）定期メンテナンス開始まで

対象条件
:   キャンペーン期間中に、初めてECOをプレイして、イベントナビゲーションのチュートリアルを全てクリアしたキャラクターをお持ちの方

| チュートリアルイベント名 | 開始方法 | クリアの確認方法 |
    | --- | --- | --- |
    | はじめてのおつかい | ［メニュー］ －［クエストナビインフォ］ －［イベントナビ］ －イベント名をクリックする | ［メニュー］ －［クエストナビインフォ］ －［イベントナビ］ －状態が［完了］になっている |
    | すてきなおくりもの |
    | だいじなおしごと |
    | たのしいなかま |
    | わくわくとたびだち |
    | ねがいとおもい |
    | たのしいおかいもの |
    | くらやみのぼうけん |
    | おしゃれとじつよう |
    | つたえたいつよさ |
    | こころざしのありか |
    | たびだちのとき |

- 本キャンペーンの「初めてECOをプレイ」とは、ECOゲームアカウントを作成し、ECOをプレイすることです。
    - ECO Light GGのお試しプレイで、対象条件を満たした後に、ECOのゲームアカウント作成とログインIDを登録した場合も本キャンペーンの対象になります。
    - ECO Light GGのお試しプレイ後、ログインIDを登録しない場合、作成したキャラクターやプレイした情報は削除されます。また、プレイ中にメンテナンス等によりゲームから切断された場合やブラウザを閉じた場合も、情報は削除されます。
    - 複数のゲームアカウント、キャラクターで対象条件を満たしても、エントリーは1つのログインIDにつき、1回限りです。

景品
:   対象条件を満たした方の中から、抽選で以下の景品をプレゼントいたします。

| 景品 | 当選人数 |
    | --- | --- |
    | ゲーム内アイテム「ダンプティースーツ」1個 | 抽選で200名様 |

- 景品の当選者は、本キャンペーンにエントリーされた全ての方の中から抽選いたします。

:   当選した方には、2017年 1月下旬 3月上旬頃、［お知らせ］で景品のアイテムと交換可能なアイテムコードをご案内いたします。※発送手続きの関係で景品の発送は3月上旬頃となります。お待たせいたしまして誠に申し訳ございません。

- ［お知らせ］は、ECO公式サイトにログイン後、左側メニューの［サポートからのお知らせ］-［投稿内容一覧]から確認することができます。
    - ［お知らせ］については、[こちら](https://secure.gungho.jp/faq/faqdetail.aspx?id=1d683bed-133e-4bf7-9717-030b9c266b8b)をご確認ください。
