#### Anemoneワールドとは？

|  |  |
| --- | --- |
|  | GM.Anemoneの世界——  Anemoneワールドが、僅か１週間だけタイニーアイランドとリンクしたようです。    Anemoneワールドでは、タイニーのフシギな力によって全ての冒険者が上位三種族となります。  GM.Anemoneだけが住む小さな世界ですが、都市攻防戦や騎士団演習が毎日開催されています。  いくつかのダンジョンへ行くことも可能です。    また、通常のワールドとは違い、職業を選んだ後でも職業を変えることができるので、今までに遊んだことのない職業で都市攻防戦や騎士団演習に参加したり、ダンジョンでたくさんのモンスターと戦ったり、存分にスキルを試すことができます。    三次職クロニクルジョブをとことん体験できるチャンスですので、皆様奮ってご応募ください。 |
