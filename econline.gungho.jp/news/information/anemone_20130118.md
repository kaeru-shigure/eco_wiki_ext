### 第7回Anemoneワールド　追加調整のお知らせお知らせ

最終更新日：2013/01/18 12:30　掲載日：2013/01/18 12:30

![第7回Anemoneワールド　追加調整のお知らせ](../../news/information/20ujqj000000fuy2-img/a1358475672641.png)

ECO運営チームです。

2013年1月15日（火）より開催しております第7回Anemoneワールドにおいて多くのご意見・ご感想をお寄せ頂き、ありがとうございます。

2013年1月18日（金）13：00より実施させて頂きます、第7回Anemoneワールド臨時メンテナンスにおいて以下のスキルについて追加調整を行わせて頂きます。

#### 対象スキルおよび調整内容

| 職業 | スキル | 調整・変更内容 |
| --- | --- | --- |
| ガーディアン | シールドインパクト | 消費SPを増加いたします。  威力を上昇いたします。  Lv3～5をモンスターに対して使用した際のステータス低下確率を上昇いたします。  ディレイタイムを短縮いたします。 |
| イレイザー | イレイザー | ディレイタイムを短縮いたします。 |
| 影縫い | PvPモード中にLv1～2を使用した際の硬直効果時間を延長いたします。  PvPモード中にLv3～5を使用した際の硬直効果時間を短縮いたします。 |
| ホークアイ | フレアショト | Lv1～4のキャストタイムを短縮いたします。  威力を上昇いたします。  追加威力補正を上昇いたします。 |
| インパクトショット | 第7回Anemoneワールドにおける調整内容から威力を上昇いたします。 |
| テンペストショット | 消費SPを増加いたします。  第7回Anemoneワールドにおける調整内容から消費する弾の数を減少いたします。 |
| フォースマスター | フォースバレット | 威力を上昇いたします。  ディレイタイムを短縮いたします。 |

| スポレーションソウル | 威力を上昇いたします。 |
| ハーヴェスト | ハーヴェスト | Lv2～5のキャストタイムを短縮いたします。  Lv2～5のディレイタイムを短縮いたします。  Lv2～5の再使用時間を短縮いたします。 |
| ストライダー | アブサードウィップ | Lv2～5のキャストタイムを短縮いたします。  Lv2～5のディレイタイムを短縮いたします。 |
| ロングレンジプルウィップ | 消費SPを増加いたします。  第7回Anemoneワールドにおける調整内容から再使用時間を短縮いたします。 |
| ピレッジアクト | 消費SPを増加いたします。  Lv1～4のキャストタイムを短縮いたします。  Lv5のキャストタイムを増加いたします。  Lv1～4のディレイタイムを短縮いたします。  Lv5のディレイタイムを増加いたします。  第7回Anemoneワールドにおける調整内容からLv4～5のアイテム取得成功率を低下いたします。  第7回Anemoneワールドにおける調整内容からLv4～5の威力を上昇いたします。 |
| ジョーカー | ジョーカースタイル | ディレイタイムを短縮いたします。  第7回Anemoneワールドにおける調整内容から威力を上昇いたします。 |
| ＤＥＭ | ソニックブレイカー | 威力を上昇いたします。 |
| クーデグラ | キャストタイムを延長いたします。  ディレイタイムを短縮いたします。  消費SPを増加いたします。  威力を上昇いたします。 |
| レガシースペル | キャストタイムを短縮いたします。 |
| インサニティヴォーテックス | キャストタイムを短縮いたします。  ディレイタイムを短縮いたします。 |

##### その他、調整内容

- アイテム「大きな矢束」を第7回Anemoneワールドに実装いたします。

※「大きな矢束」はスキル「投擲武器作成」Lv4を使用して「木の矢」300本と「ひも」2本から作成することができます。  
  ※「大きな矢束」「矢束」「大きな弾倉（実包）」「弾倉（実包）」「大きな弾倉（弱弾）」「弾倉（弱弾）」については重量と容量の減少を行いますが、第7回Anemoneワールドでは対応は見送らせて頂きます。  
  ※AnemoneワールドではNPCより「大きな矢束」の購入が可能です。
- NPC「武器屋」にアイテムを売ろうとした際に、身動きが取れなくなってしまう不具合を修正いたします。
