### 8月25日（木）実装の新スキルについて、第三回「Anemoneワールド」からの変更点お知らせ

![更新](../../common/_wr/img/icon_update.gif) 最終更新日：2011/08/30 14:00　掲載日：2011/08/25 11:00

2011年6月20日（月）〜2011年6月27日（月）の期間に実施いたしました第三回「Anemoneワールド」での新スキルについて、ご参加いただきました皆様よりいただいたご意見を踏まえ、下記のスキルについて調整いたしましたことをご報告いたします。

|  | スキル名称 | 変更点 |
| --- | --- | --- |
| 1 | 神速斬り | 再使用時間を設定しました。  「ストライクブロウ」の習得レベルにより消費SPが減少するようにしました。  レベルの上昇により、スタン効果時間が伸びるように変更しました。  モンスターとの距離によるダメージ減少を削除しました。  命中率の上昇に関するステータスをINTからDEXに変更しました。 |
| 2 | リムーブウエポン | 物理攻撃力低下値を下方調整しました。  魔法攻撃力低下を追加しました。  ダメージ量を上方調整しました。 |
| 3 | ブロッキング | 消費SPをレベル毎に設定しました。  スタン効果時間を上昇調整しました。  スタン効果とダメージ減少が個別に発動するように変更しました。  ダメージ減少確率を上方調整しました。  ダメージ減少効果を上方調整しました。  「シールドマスタリー」習得レベルによりダメージ減少効果が上昇するようにしました。  遠距離攻撃、魔法攻撃でもスキルが発動するようにしました。 |
| 4 | ストロングボディー | 消費SPが増加しました。  効果時間を上方調整しました。  術者に対する基本防御力の増加を上方調整しました。  パーティーメンバーに対する基本防御力の増加を下方調整しました。  スキルの効果に追加防御力の増加を追加しました。 |
| 5 | サーチウィークポイント | 術者に対するクリティカルの増加を上方調整しました。  スキルの効果に命中率の増加を追加しました。 |
| 6 | ポイズンマスター | 効果時間の増加を上方調整しました。  スキルの効果に「豪腕毒」の攻撃力上昇を追加しました。  スキルの効果に「硬化毒」の防御力上昇を追加しました。" |
| 7 | デスバインド | スキル名称を「影縫い」に変更しました。  演習時使用不可に変更しました。  再使用時間を短縮しました。  範囲にスキル効果を発揮するように変更しました。 |
| 8 | フォトンランチャー | スキルのエフェクトを変更しました。  [2011年8月30日（火）追記]  リフレクション、リフレクションウォール、リフレクトシールドで反射されないように修正しました。 |
| 9 | アドバンスアビリティー | エクヴィリーと平行して使用している場合に、効果が高いほうを優先するように変更しました。 |
| 10 | 死鎌演舞 | ダメージ量を上方調整しました。  スキルのエフェクトを調整しました。 |
| 11 | フェンリル召喚 | 演習時使用不可に変更しました。  MPを消費するように変更しました。  Lv1の即死確率を上方調整し、EXPを取得出来ないように変更しました。  Lv1を使用せずにLv2、Lv3を使用できるように変更しました。  Lv1を使用していた場合、Lv2、Lv3の効果が上昇するように変更しました。  スキルエフェクトを大きくしました。  Lv3を使用した際、必ず命中するように変更しました。  Lv2とLv3の効果量を調整しました。  フェンリル召喚のLv2使用時にSPも回復するように変更しました。 |
| 12 | ウィンドエクスプロージョン | 消費MPを増加しました。  スキルエフェクトを小さくしました。 |
| 13 | パクティオ | 再使用時間を短縮しました。  効果時間を上方調整しました。  契約した属性の与ダメージ補正値を上方調整しました。  契約していない属性の被ダメージ補正値を下方調整しました。 |
| 14 | ディバインブレーカー | ダメージ量を上方調整しました。  ディレイ時間を短縮しました。 |
| 15 | リカバリー | キャスト時間を短縮しました。  ディレイ時間を短縮しました。  回復力を上方調整しました。  消費MPを減少しました。 |
| 16 | ハーヴェストマスター | 「ブリックルシード」に対して術者の能力が反映される効果を追加しました。  黒薔薇の棘のダメージ補正が上昇する効果を追加しました。  「マリオネットオーバーロード」の時間が増加する効果を追加しました。  「ネガティブハーヴェスト」の状態異常効果時間を上昇する効果を追加しました。  「ハーヴェスト」の再使用時間が短縮される効果を追加しました。  「ハーヴェストマスター」対象スキルの消費SPを減少する効果を追加しました。  マリオネットキング召喚系のスキル威力上昇率を調整しました。 |
| 17 | 植物寄生 | スキル名称を「ツイステッドプラント」へ変更しました。  消費MPを増加しました。  演習時使用不可に変更しました。  スキルを重ねて発動することが出来ないように変更しました。  憑依時使用不可にしました。 |
| 18 | レールガン | 使用する触媒を「鉄くず」から「レールガン専用弾」へ変更しました。  「レールガン専用弾」を作成することが出来るレシピを追加しました。  「弾倉（レールガン専用弾）」を特定のNPCにて販売します。  必ず命中するように変更しました。  モンスターとの距離によるダメージ減少を削除しました。 |
| 19 | サテライト | 演習時使用不可に変更しました。  サテライトが術者から引き継ぐ能力を上方調整しました。  召喚されたサテライトのスキルに「スタンブロウ」を追加しました。 |
| 20 | トレジャーマスター | 「ファシネイショントラップ」の威力補正が上昇する効果を追加しました。  「グラストラップ」のダメージ量上限が上昇する効果を追加しました。  消費SPを減少する効果が適用されるスキルを調整しました。 |
| 21 | 知識の書 | スキルの効果範囲内にいるパーティーメンバーにも効果が発動するように変更しました。 |
| 22 | 鞄の達人 | 鞄装備時のダメージ量補正を上方調整しました。  「ビートスマッシュ」の攻撃回数追加の効果を上方調整しました。 |
| 23 | 神の加護 | 消費SPを増加しました。 スキルにより上昇するステータス値を上方調整しました。 |
| 24 | 道化師の魂 | 習得レベルを「40」から「25」に変更しました。 |

今後も皆様のご期待におこたえできるように開発を続けてまいりますので、ご意見・ご要望をWEBヘルプデスクよりご投稿ください。
