### 5月19日（木）実装の新スキルについて、第二回「Anemoneワールド」からの変更点お知らせ

![更新](../../common/_wr/img/icon_update.gif) 最終更新日：2011/05/20 17:50　掲載日：2011/05/12 11:00

2011年3月8日（火）〜2011年3月15日（火）の期間に実施いたしました 「Anemoneワールド」での新スキルについて、ご参加いただきました皆様より いただいたご意見を踏まえ、下記のスキルについて調整いたしましたことを ご報告いたします。

[2011年5月20日更新]スキルに「アースクライ」を追記しました。

###### ■3次職クロニクルジョブ新スキルのAnemoneワールドからの変更点

|  |  |  |
| --- | --- | --- |
|  | スキル名称 | 変更点 |
| 1 | プレッシャー | 攻撃・詠唱速度減少の効果を上方調整しました。  設置数に制限を設けました。 |
| 2 | サヴァイブ | 発動条件を調整しました。  ノックバックさせていた距離を調整いたしました。  サヴァイブ発動時、不屈の闘志が同時に発動するように変更しました。 |
| 3 | シールドインパクト | スキル効果範囲を調整しました。  ターゲットの仕様を変更しました。  スタン確率を上方調整しました。  再使用時間を短縮しました。  詠唱時間を短縮しました。  憑依時使用不可に変更しました。 |
| 4 | スパイラルスピア | レベル3、レベル5のスキル効果範囲を拡大しました。 |
| 5 | 悪鬼 | ダメージ量を上方調整しました。  新しい状態異常効果を追加しました。 |
| 6 | マーシレスシャドウ | ダメージ量を上方調整しました。  再使用時間を短縮しました。 |
| 7 | テンペストショット | レベル3、レベル5のスキル効果範囲を拡大しました。  憑依時使用不可に変更しました。 |
| 8 | フレアショット | 消費SPが増加しました。  状態異常発生率を上方調整しました。  ダメージ量を上方調整しました。 |
| 9 | フリークブラスト | 消費MPを増加しました。  ダメージ量を上方調整しました。  スキル効果範囲を拡大しました。  詠唱時間を増加しました。  新しい状態異常効果を追加しました。 |
| 10 | フォースバレット | ダメージ量を上方調整しました。  ターゲット消滅時、フォースバレットも消滅するようにしました。 |
| 11 | エレメンタルブレット | 消費MPを増加しました。  ダメージ量を上方調整しました。  レベル4、レベル5のディレイ時間を短縮しました。  スキル動作中に移動することができるようにしました。 |
| 12 | エンジェルリング | ディレイ時間を増加しました。 |
| 13 | アビス | 消費MPを増加しました。  ダメージ量を上方調整しました。  再使用時間を短縮しました。  演習時使用不可にしました。 |
| 14 | ロックフォール | ダメージ量を上方調整しました。 |
| 15 | クリエイトネコグルミ | 再使用時間を短縮しました。  ネコグルミの動作を変更しました。  レベル2、レベル3の際にプレイヤーから引き継ぐ能力を上方調整しました。 |
| 16 | ブリックルシード | ダメージ量を上方調整しました。  状態異常成功率を上方調整しました。 |
| 17 | リージョンビバーク | 設置数に制限を設けました |
| 18 | ジョーカースタイル | ダメージ量を上方調整しました。  Ｌｖ3の使用可能武器を追加しました。  状態異常成功率を上方調整しました。 |
| 19 | アースクライ | 効果時間を調整いたします。  消費MPを変更いたします。 |

今後も皆様のご期待におこたえできるように開発を続けてまいりますので、ご意見・ご要望をWEBヘルプデスクよりご投稿ください。
