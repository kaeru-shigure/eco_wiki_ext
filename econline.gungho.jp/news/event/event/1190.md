### ECOハートフルステージ2010開催決定！イベント

![更新](../../../common/_wr/img/icon_update.gif) 最終更新日：2010/04/21 　掲載日：2010/04/19

|  |  |
| --- | --- |
| ECO運営チームです。    豪華声優陣の参加によって大反響を呼んでいる、ECOドラマCD Vol.2の発売を記念して、4月29日（木）にオフラインイベント「ECOハートフルステージ2010」の開催が決定いたしました！    場所は、秋葉原PCゲームフェスタ（4月29日〜5月9日）が開催されているベルサール秋葉原の地下一階イベントステージ。    そしてなんと！イベントゲストとして、ECOドラマCDにおいてティタ役を務めた能登麻美子さんと、ネコマタ・桃役を務めた井口裕香さんが登場いたします！ |  |

![](../../../news/event/event/20ujqj00000003a9-img/photo1.jpg)

気になるイベントの内容ですが、まずイベントの目玉としてゲストのお二人により、この日、この会場でしか聞けないオリジナルストーリーの生アフレコを行ないます！ティタとネコマタ・桃によって一体どんな物語が繰り広げられるのか…、これは絶対に聞き逃せません！

他にもECO○×クイズや、ドラマCDについての面白トークなどお楽しみ要素が満載です！

また当日、会場一階にてECOドラマCD Vol.1&Vol.2の販売も行なっております！  
是非、4月29日はベルサール秋葉原へお越しください！

###### 【会場】

ベルサール秋葉原　B1Fホール

###### 【開催日時】

4月29日（木）開場15：30〜開演16：00〜（17：00終了予定）  
**※当日、10：30より整理券を配布いたします。**[2010/4/21更新]  
※会場や、前のイベントの都合により、時間は変更となる可能性がございます。あらかじめご了承ください。
