### 3月26日に都市攻防戦が拡張＆CPショップにも新作が！イベント

最終更新日：2010/03/05 　掲載日：2010/03/05

ECO運営チームです。

3月26日のアップデートで、都市攻防戦が拡張されます！  
これにより新しい戦場となるMAPや、  
新モンスター、さらには新しいCPショップのアイテムまで実装されます！  
これまで以上に強力なＤＥＭ達との戦いを勝ち抜き貴重なアイテムを手に入れましょう。

###### ■新戦場MAP

|  |  |
| --- | --- |
|  |  |

新しい戦場MAPはなんと、ドミニオン世界軍艦島地下に広がる「地下浄水施設」！   
レジスタンス達が結集する最前線基地のすぐ目の前に広がるMAPです。 ここで、より強力なＤＥＭ達を迎え撃ってください！

###### ■新モンスター

|  |  |  |
| --- | --- | --- |
|  |  |  |

ウェストフォートに進軍してくるＤＥＭ達よりも、さらに強力なＤＥＭ軍団！  
より巨大になった者や小型になった者、そして強力なチャンプＤＥＭやどこかで見たような姿のＤＥＭも……？

###### ■新CPショップラインナップ

|  |  |
| --- | --- |
|  |  |
|  |  |

都市攻防戦の拡張と共にCPショップのアイテムが多数追加されます！   
新実装されるアイテムに御期待ください！
