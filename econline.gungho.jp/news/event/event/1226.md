初夏に入り、いよいよ本格的に暑くなる季節がやってきました！

太陽のタワーを駆け抜けましょう!!行く手を阻む門番をかわして進み、皆でゴールを目指すのだ!!  
アツく盛り上がるＥＣＯでお楽しみください。

**■ タイニーアイランドへの行き方**

アップタウンの南側にいるタイニー（座標：ｘ127,ｙ153）に話しかけ、「いってみたい」を選択するとタイニーアイランドへ行けます。

**■ ウテナ湖への行き方**

東アクロニア平原の南に進むと、次のマップがウテナ湖です。

|  |  |
| --- | --- |
| ◎貸し衣装イベント  「愛の花束」を持って、タイニーアイランドにいる休憩しているタイニーに話しかけると衣装を借りることができます。  華やかな衣装に着替えてから他のイベントに進もう！ | ◎ポスター交換イベント  ダウンタウンにいる気紛れな絵描きから季節限定ポスターを手に入れよう！ |
| ◎アクロニア大聖堂移動イベント  アップタウンにいるキュー（座標：127,139）に話しかけるとアクロニア大聖堂へ移動できます。 | ◎絆イベント  「絆のかけら」を集めて友達との絆の証「絆証明書」を手に入れよう！集めた絆のかけらの数に応じて、さまざまな家具アイテムと交換可能です！ |
| ◎バレンタイン聖堂移動イベント  タイニーアイランドのレインボー発生装置に話しかけるとバレンタイン聖堂へ移動できます。 | ◎音楽祭イベント  バレンタイン聖堂にいる指揮カエルのお願いを聞いて、結婚式を成功させよう！ |
| ◎帰ってきたフシギ団イベント（1）  雨の降るウテナ湖にフシギ団の幹部パケルが登場！  雨をやませる儀式に必要な祭器を、助手の女の子・ポロンのところまで届けてくれる冒険者を探しているようですが… | ◎帰ってきたフシギ団イベント（2）  ポロンのいる太陽のタワーの屋上までには多数の門番が！　タワー内ではHPの最大値が1になってしまいます。  行く手を阻む門番をよけてかわすか、攻撃で突破するか!? |
| ◎帰ってきたフシギ団イベント（3）  そして、太陽のタワー頂上に眠る太陽の精霊とは!?   復活の儀式中のポロンに太陽の祭器を渡し、儀式がうまくいくように手伝ってあげましょう。 | ◎帰ってきたフシギ団イベント（4）  ポロンの儀式がうまくいくと「勝利のユニフォーム」「チアガールのユニフォーム」「勝利のボール」など、豪華にたくさんのアイテムを入手できます！  是非みんなでゲットしよう！ |
