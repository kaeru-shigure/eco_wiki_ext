##### 二時間目：まごころのおくりもの

今月のイベントをクリアすると、「この先『○○』に注意！」がもれなくもらえます！

さらに、イベントをクリアした後にアップタウンにいる守護魔・キサラギに話しかけると  
ＥＸイベントチケット5枚と交換で、アクセサリー「恋の魔法陣」を手に入れることが出来ます！  
（○で隠された部分には、あるモンスターの名前が入ります。実際にゲームの中で確かめましょう！）

全ワールド

###### 今月から始めても大丈夫！「生徒手帳」＆「ＥＸイベントチケット」

アルマたちと絆の樹は、通年を通して緩やかなつながりはありますが、  
それぞれが一話で完結するイベントです。

クリアしていない過去のイベントがあって「今から始めても……」と、思ってる人も大丈夫！  
イベントをクリアした人から「生徒手帳」を譲ってもらうか、  
ＥＣＯくじで手に入る「ＥＸイベントチケット」があれば、  
アミス先生にアルマたちを呼んでもらう事ができます。

※「生徒手帳」は月毎にアイテムが異なります。  
    例えばダークフェザー・アルマを呼ぶ場合は「１月の生徒手帳」が必要です。  
※既に教室にいるアルマ・守護魔ハート・ネコマタハートを呼ぶことはできません。

また、教室にいるアルマたちからは、過去のイベントの出来事を教えてもらう事ができます。

###### 生徒手帳をもらう

イベントをクリアしている人は翌月以降、アミス先生の空飛ぶ教室にいるアルマに話かける事で、生徒手帳を２つもらう事ができます。

※ダークフェザー・アルマから貰える生徒手帳は「１月の生徒手帳」です。

![生徒手帳をもらう](../../../news/event/event/20ujqj000000526s-img/student_handbook.jpg)

###### アミス先生の空飛ぶ教室にアルマを呼ぶ

生徒手帳を所持した状態でアミス先生に話かけると、生徒手帳に対応したアルマを呼ぶ事ができます。  
例えば「１月の生徒手帳」を所持した状態でアミス先生に話かけると、  
ダークフェザー・アルマを呼ぶ事ができます。

ＥＸイベントチケットを所持した状態でアミス先生に話かけると、これまでに登場した好きなアルマを  
ＥＸイベントチケット１枚につき１人呼ぶ事ができます。

![アミス先生の空飛ぶ教室にアルマを呼ぶ](../../../news/event/event/20ujqj000000526s-img/call_aruma.jpg)

###### アルマにネコマタハートまたは守護魔ハートを呼んでもらう

アミス先生の空飛ぶ教室にいるアルマと対応する生徒手帳を持った状態、  
または、ＥＸイベントチケットを所持した状態でアルマに話かけると、  
アルマが一緒に活躍したネコマタハートまたは守護魔ハートを呼んでくれます。

例えば「１月の生徒手帳」を所持した状態でダークフェザー・アルマに話かけると、  
シャノワールか守護魔・ムツキを教室に呼んでくれます。

![アルマにネコマタハートまたは守護魔ハートを呼んでもらう](../../../news/event/event/20ujqj000000526s-img/call_nekomata.jpg)

「生徒手帳」とは……？

例えば２月のイベントをクリアした人が、３月のイベントが始まった時点で、  
バウ・アルマに話しかけると「２月の生徒手帳」が２つもらえます。  
生徒手帳は各アルマから１回のみ入手が可能です。（キャラクター毎）

今月までに生徒手帳が作成可能なアルマ

ダークフェザー・アルマから「１月の生徒手帳」  
バウ・アルマから「２月の生徒手帳」が、もらえます。

「ＥＸイベントチケット」とは……？

ECOくじの副賞として入手できるアイテムです。

現在、教室に呼び出せるアルマ・守護魔ハート・ネコマタハート

| 関連アイテム | 呼び出せる生徒たち | |
| --- | --- | --- |
| １月の生徒手帳 | ダークフェザー・アルマ | 守護魔・ムツキ、シャノワール |
| 生徒手帳の代わりにEXイベントチケットを使用すると、  １枚につき上記生徒の中から一人呼び出す事ができます。 | | |

※守護魔ハートとネコマタハートは、アルマが呼び出された状態でなければ、教室に呼ぶ事ができません。
