#### ECO祭2013ファッションコンテスト「エコ・コレ」

##### 内容

ECOのゲーム内で、キャラクターをおしゃれにコーディネートしたスクリーンショットを撮影し、「ECO祭2013ファッションコンテスト「エコ・コレ」」フォームから応募してください。  
応募作品を、ECO運営チームが選考し、入賞作品を決定させていただきます。入賞作品は、「ECO祭2013」で同じ装備をしたキャラクターによるファッションショーを開催し、発表させていただきます。  
また、入賞作品の中から、「ECO祭2013」会場での投票や生放送による投票をもとに、男性部門・女性部門各1名の最優秀賞を決定いたします。  
入賞者には、各賞に応じて、ECOゲーム内アイテムをプレゼントいたします。

###### 応募期間

2013年10月24日（木）15：00～2013年11月14日（木）10：00

###### 応募条件

・ECOのキャラクターをお持ちの方

###### 各賞および賞品について

【最優秀賞】（男性部門・女性部門各1名）  
・ECOゲーム内アイテム「フシギなヘアカタログ3」  
【入賞】（男性部門・女性部門各5名）  
・ECOゲーム内アイテム「紹介状」  
※入賞賞品は、以下の中から、いずれか1つを応募の際に選択していただきます。

| 入賞賞品の紹介状 | | | | |
| --- | --- | --- | --- | --- |
| 黒き翼の紹介状 | ムツキの紹介状 | 忠犬の紹介状 | キサラギの紹介状 | 萌芽の紹介状 |
| ヤヨイの紹介状 | あやかしの紹介状 | ウヅキの紹介状 | やんちゃ娘の紹介状 | サツキの紹介状 |
| 潮風の紹介状 | ミナヅキの紹介状 | 南極の紹介状 | フミツキの紹介状 | アラビアンナイトの紹介状 |
| ハヅキの紹介状 | おかしな紹介状 | キクヅキの紹介状 | 使い魔の紹介状 | カミア・カンナの紹介状 |
| スチームパンクの紹介状 | シモツキの紹介状 | 闇夜の紹介状 | ロウゲツの紹介状 |  |

※【最優秀賞】を受賞した方は、【最優秀賞】と【入賞】両方の賞品を受け取ることができます。

###### 応募方法

[「ECO祭2013ファッションコンテスト「エコ・コレ」」フォーム](https://entry2.gungho.jp/entry/entrance.aspx?entid=7c9246c3-3d18-4b71-a195-f0d0a58df6be)に必要事項を記入し、スクリーンショットを添付して応募してください。

**【必要事項】**

- プレイワールド名
- エントリーネーム（30文字以内）
- キャラクターの性別、種族
- 装備しているアイテム（正式名称で入力してください）  
  ペットアイテムは選考対象外となるため、記入していただく必要はありません。
- コメント（500文字以内）
- スクリーンショット画像（画像形式JPG：512KB以内）
- 入賞時に希望する賞品（ご希望の「紹介状」を選択してください）
- 氏名、住所、電話番号、メールアドレス（賞品の発送や応募完了メールの送信に使用いたします）

- ※応募内容の確認、変更、取り消しはできません。
- ※応募内容は、「ECO祭2013」で発表したり、ECO公式サイト等で公開したりする場合があります。
- ※ゲームアカウントは公開いたしません。
- ※スクリーンショットでキャラクターが装備しているアイテムと応募フォームに入力いただいたアイテムに相違がある場合、スクリーンショットのアイテムを優先させていただきます。
- ※ご自身で作成・利用しているキャラクターのスクリーンショットをご応募ください。公開前、公開後にかかわらず、ECO運営チームが第三者のキャラクターであると判断した場合、応募を無効とさせていただきます。
- ※応募にあたり、ログインは必要ありません。
- ※スクリーンショットをトリミングする必要ありません。

###### 応募作品の選考および結果発表について

Step 1 :　1次選考  
応募いただいた作品をECO運営チームが選考し、入賞作品の装備を「ECO祭2013」のファッションショーで発表いたします。

Step 2 :　ユーザー投票  
「ECO祭2013」会場内投票とニコニコ生放送のアンケートで、ユーザーの皆様に投票をしていただきます。

Step 3 :　結果発表  
ユーザーの皆様からの投票結果をもとに、「ECO祭2013」のイベント中に、各賞の発表をいたします。

###### 賞品の配布について

入賞された方には、応募の際に入力されたご住所あてに、賞品アイテムのアイテムコードをお送りさせていただきます。  
※賞品の発送は、2013年12月下旬ごろを予定しております。
