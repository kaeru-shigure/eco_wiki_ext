### ワンタイムパスワード（OTP）付BitCash発売！イベント

最終更新日：2010/04/09 　掲載日：2010/04/09

![ワンタイムパスワード（OTP）付BitCash発売！](../../../news/event/event/20ujqj00000002mx-img/main.jpg)

ECO運営チームです。

全国のコンビニエンスストア「ファミリーマート」にあるマルチメディア端末「Famiポート」にて期間限定でワン タイムパスワード（OTP）付BitCashを販売することとなりました。

![Famiポート](../../../news/event/event/20ujqj00000002mx-img/famiport.jpg)

ワンタイムパスワード（OTP）付BitCashは、ガンホーコインが購入できる「BitCash2,000クレジット」とあなたの大切なアカウントを守る「OTP利用権（365日）チケット＆携帯OTPアプリダウンロードクーポン」がセットになった大変お得な商品です。  
さらに！このセットには超目玉ゲーム内アイテム「宙飛ぶ南京錠」がついてきます！  
「宙飛ぶ南京錠」はその名の通り、宙に浮いて飛ぶ南京錠で、騎乗ペットアイテムとなっておりますので、あなたを乗せて目的地へと運んでくれます。  
特徴的なその見た目は、周りの人の注目を浴びること間違いなし！

2010年4月13日（火）7：00〜5月31日（月）23：59

###### ■販売価格

3,000円（税込)

8,000セット（限定）

※「BitCash2,000クレジット」は、ガンホーゲームズでのガンホーコインの購入にのみご利用いただけます。  
※ゲーム内アイテム「宙飛ぶ南京錠」はアイテムコードをゲーム内の「アイテムチケット交換機」で使用することで、入手することができます。

**Famiポートで商品を購入する**

全国のファミリーマートにあるマルチメディア端末「Famiポート」で、購入手続きを行います。  
Famiポートで、プリペイドカード→電子マネー→エミル・クロニクル・オンライン→ワンタイムパスワード(OTP)付BitCashの順に進みます。

最後にFamiポートから出てきたレシートをレジカウンターに渡し、番号通知票を発券してもらい、購入します。  
Famiポートの使い方は、ファミリーマート公式サイトの[「Famiポートの使い方」](http://www.econline.jp/redirect/re.php?a=http://www.family.co.jp/famiport/howto/)をご覧ください。

**チケットの使い方**

|  |  |
| --- | --- |
| ガンホーゲームズのアトラクションセンターに  ログインしていただき、  「ガンホーコイン購入」をクリック。  コインの種類「2,000コイン」と  決済種別「BitCash」を選択したのち、  番号通知票の「ひらがなID」を入力すると  ガンホーコインが購入できます。  ガンホーコインについては[こちら](https://secure.gungho.jp/faq/faqdetail.aspx?id=4c956a0e-d8ac-4747-ad5d-8c70ea221414)。 |  |
| **「BitCash会員サービス」にログイン**    [BitCash公式サイト](http://www.econline.jp/redirect/re.php?a=http://www.bitcash.jp/)にアクセスし、  会員サービスにログインしてください。  会員登録は無料 です。 |  |
| **ひらがなIDを登録する**    ログインしたのち、「MyBitCash」欄に、  ガンホーコインのお支払いにご利用いただいた  「ひらがなID」を入力していただくと、  登録が完了します。 |  |
| **アイテムチケットIDをゲット！**    登録が終わると、MyBitCashの一覧に  ひらがなIDが表示されますので、  「CPコード確認」ボタンをクリックしてください。  「空飛ぶ南京錠」のチケットIDが表示されます。  以降はCPコード確認ボタンを押すと、  いつでもチケットIDが確認できます。  「BitCash」の残高があるひらがなIDは  チケットIDが表示されません。 |  |
| **アイテムと交換する**ゲーム内アイテム「宙飛ぶ南京錠」はアイテムコードをゲーム内の「アイテムチケット交換機」で使用することで、入手することができます。 | |
