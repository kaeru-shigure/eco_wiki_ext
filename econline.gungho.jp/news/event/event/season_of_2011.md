今年も楽しいお正月がECOにやってきます。町中はお正月模様に染まり、おめでたい新年のムードに彩られます。

今年は兎年。干支である兎にちなんだアイテムも登場します。

###### ■イベント内容

各地に鳥居や神社が出現します。

他にも様々なイベントが各地で行われておりますので、是非ご参加ください。

###### ■「ＥＸイベントチケット」でゲーム内イベントをさらに楽しもう！

年末から１月にかけて、アップタウンの白の聖堂裏手に「和風の家具職人」という人物がやってきています。  
どうも、この寒空の下なにやら途方にくれている様子……。  
もし「ＥＸイベントチケット」を持っていたら、彼に話しかけてみましょう。  
「ＥＸイベントチケット」１枚と引き換えに「古風なふすま」か「おしゃれなふすま」のどちらかが１枚手に入ります。  
今回はまとめて交換も可能になっていて、なんと「ＥＸイベントチケット」３枚でふすまを４つ手に入れることができますよ！

|  |  |
| --- | --- |
| 途方にくれている家具職人に話を聞いてみよう。 | オーソドックスな「古風なふすま」（２枚） |
| デザイン重視の「おしゃれなふすま」（２枚） | 合計４枚で、こんな組み合わせもできちゃう。 |
