ECO運営チームです。  
待ちに待った女の子のお祭り「ひなまつり」、そして2月のバレンタインに引き続き  
「ホワイトデー」の到来です。  
アクロポリスシティはあっちもこっちも華やかな空気があふれ、様々な人がやってきます。  
また、南ウテナ開拓地周辺にも、何やら妙な人物（？）がやって来ているようです。

おめかしを楽しんだり、チョコのお返しを期待したり。  
とはいえ、アクロニア平原には何やら不穏な影も……？

今月もたくさん開かれている様々な催しを、仲の良いお友達や、  
いつものリングの仲間たちと一緒に遊んで、楽しい３月を過ごしましょう！

#### ホワイトデー＆ひなまつり2012！

今年の新アイテム「ストライプショートパンツ（茶・水・黒・桃） 」は、南ウテナ開拓地の海側にやってきた訪問者「勝ち組インスマウス」の悩みを解決してあげることで、どれか一つ手に入りますよ！

###### ■イベント内容

・仕立屋アマス、アップタウンにアラワル

有名なデザイナー、「仕立屋アマス」（X128,Y139）がアップタウンにやって来ています。

ですが、彼女は素材不足に悩まされているようです。

もし、素材を提供することができれば素敵な衣装を手に入れることができるかも……？

・シアターがひな壇仕様に変更

ダウンタウンのシアターに、この時期だけひな壇が設置されます。

おめかしをして、皆で記念撮影などいかがでしょう？

|  |  |
| --- | --- |
| 仕立て屋アマスと、彼女の相棒しゃるろっとちゃん。 | シアターのスクリーン手前に大きなひな壇が登場！ |

・家具職人の家具リサイクル  
各地にいる「家具職人」の所で、ひな祭りにちなんだ家具アイテムを交換することができます。  
何を何と交換できるかは、日によって違うのでいろいろ試してみましょう！

・ホワイトデーイベント  
バレンタインにアップタウンのティタ、マーシャ、ルルイエチョコケーキをもらった人は  
お返しにマシュマロをプレゼントしてみてはいかがでしょう？  
また、バレンタインにタイタス、エミル、ベリアルにチョコケーキをプレゼントした人は、  
彼らからお返しがもらえるかもしれません。

|  |  |
| --- | --- |
| 各地の家具職人のところで、家具アイテムの交換ができるぞ！ | チョコのお返し、ちゃんとしました？ |

・「預かり物の手紙」のお話  
バレンタインイベントで「預かりものの手紙（イベント）」を預けられた人は、  
この時期にダブルクリックで「預かりものの手紙（イベント）」を使ってみると……？  
※誰かに憑依されている場合、憑依している場合はこのアイテムを使用することができません。  
※エミル世界でのみ使用することができます。

・奇妙な旅人、勝ち組インスマウス！  
南ウテナ開拓地に、珍しいものを探しに「勝ち組インスマウス」（X39.Y233）がやって来ています。  
なんでも、モテてモテてしょうがないというこの人物は、ありきたりではない  
変わったものを探しているのだとか……？  
もしかしたら、ウテナ湖のどこかにいる旅人「ラロ」がカギを握っている！……かもしれません。

|  |  |
| --- | --- |
| 「預かりものの手紙（イベント）」と使ってみると、そこには……？ | 南ウテナ開拓地にやってきた奇妙な客人。何を探しているのかな？ |

・モテナイ男たちの怨念  
アクロニア平原に、「モテナイ男達の怨念」が出現しています。  
バレンタインにチョコをもらえなかった男たちの悲しい想いを収めるために、  
アップタウンにいる「ユノ」（X122,Y138）」に話を聞いてみましょう。  
多忙なユノに変わって、愛の力（ユノから受け取れるラブレター）で「モテナイ男達の怨念」を鎮めれば、  
ユノから様々な報酬がもらえます！  
※バレンタインのあるイベントで手に入る「ハートフルボゥ」を使うこともできます。

|  |  |
| --- | --- |
| 忙しそうな天使のユノ。ラブレターが足りなくなったら、彼女から買うことができるぞ。 | チョコをもらえなかった男達の怨念が集まって生まれた悲しいモンスター。愛の力で鎮めよう！ |
